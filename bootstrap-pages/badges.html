Badges com Bootstrap 4

Badges são usados para trazes uma informação ou complemento adicional para um elemento. Normalmente um badge corresponde ao tamanho do elemento pai.

Para criar uma badge usamos normalmente a classe .badge em um elemento <span>, em conjunto com outras classes como badge-* para dar cores aos elementos.

Mostrar Código

<h1> Novo carro <span class="badge badge-danger"> NOVO </span> </h1>

Novo carro NOVO



Podemos usar todas as cores pré-definidas no B4 para criar o badges. As cores podem ser: .badge-primary .badge-secondary .badge-success .badge-danger .badge-warning .badge-info .badge-light .badge-dark
