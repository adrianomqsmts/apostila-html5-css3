Barras de Progresso com Bootstrap 4

Para criar uma barra de progresso padrão usamos um elemento pai com a classe .progress e um elemento filho com a classe .progress-bar. Para especificar a quantidade preenchida de uma barra, devemos usar CSS e através do atributo style definir a largura (width) em porcentagem.

Mostrar Código

<div class="progress">
<div class="progress-bar" style="width:70%"></div>
</div>



<div class="progress">
<div class="progress-bar" style="width:50%">50%</div>
</div>

50%


<div class="progress">
<div class="progress-bar" style="width:30%">Barra de 30%</div>
</div>

Barra de 30%


Para definir a altura (height) da barra devemos especificá-la usando CSS. OBS: DEVE-SE definir tanto para o filho, quanto para o pai.

<div class="progress" style="height: 30px;">
<div class="progress-bar" style="width:30%;height:30px">Barra Alta 30%</div>
</div>

Barra Alta 30%

.progress-bar-striped

A classe .progress-bar-striped adiciona listras na barra de progresso.

Mostrar Código

<div class="progress">
<div class="progress-bar progress-bar-striped" style="width:30%"></div>
</div>

<div class="progress">
<div class="progress-bar bg-warning progress-bar-striped" style="width:40%"></div>
</div>

<div class="progress">
<div class="progress-bar bg-danger progress-bar-striped" style="width:50%"></div>
</div>



.progress-bar-animated

A classe .progress-bar-animated adiciona uma animação sobre as listras de um barra de progresso.

Mostrar Código

<div class="progress">
<div class="progress-bar bg-danger progress-bar-striped progress-bar-animated" style="width:50%"></div>
</div>



Múltiplas barras de progresso

Podemos definir em um única linha de progresso várias partes coloridas. Segue o código abaixo.

Mostrar Código

<div class="progress">
<div class="progress-bar bg-danger" style="width:20%">Barra 1</div>
<div class="progress-bar bg-success" style="width:20%">Barra 2</div>
<div class="progress-bar bg-warning" style="width:40%">Barra 3</div>
</div>
