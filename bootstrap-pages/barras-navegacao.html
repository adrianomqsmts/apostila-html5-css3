Barras de Navegação (navbar) com Bootstrap 4

Para criamos uma barra de navegação padrão B4 precismos primeiro definir definir um elemento semântico <nav> com as classes .navbar e .navbar-expand que contém uma lista <ul> com a classe .navbar-nav, seguida pelos itens <li>, cada um deles, com uma classe .nav-item, e por fim cada link <a> definido com uma classe .nav-link.

Podemos usar os links <a> em conjunto com as classes .active para definir o link atual ou a classe .disabled para desabilitar algum link.

Mostrar Código

<nav class="navbar navbar-expand">
<ul class="navbar-nav">
<li class="nav-item">
<a class="nav-link" href="#">Link1</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link2</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link3</a>
</li>
</ul>
</nav>

collapse: Barra de navegação com menu hambúrguer

Vamos seguir alguns passos para a criação de um menu contraído com o ícone conhecido como hambúrguer.

  • 0 º - Antes deve definir no elemento <nav> a classe .navbar-expand-xl|lg|md|sm que irá definir em qual tamanho de tela (xl-lg-md-sm) a barra de navegação deve ser recolhida.
  • 1º - Definir um elemento link <a> ou botão <button> fora do elemento da lista <ul> e dentro do elemento <nav>.
  • 2º - Dentro da tag inicial do elemento <a> ou <button>.
    • Colocar a classe .navbar-toggler.
    • Colocar o atributo data-toggle="collapse".
    • Colocar o atributo data-target="#id-div" no caso de um elemento <button> ou o um atributo href="#id-div" no caso de um elemento <a> especificando o identificador únido da div que irá circundar o elemento <ul>
  • 3º - Entre as tags do elemento <a></a> ou <button></button> inserir um elemento <span> com a classe .navbar-toggler-icon
  • 4º - Circundar a lista <ul> com o elemento <div>.
  • 5º - Dentro da tag inicial do elemento <div> inserir:
    • As classes .collapse e .navbar-collapse
    • Inserir o atributo identificador id="id-div" com um valor único que deve ser IGUAL ao especificado no atributo data-target="" ou href="" nos passos anteriores.
  • 6º - Aplicar um background de fundo para personalizar um pouco mais a barra de navegação com as classes bg-*.
  • 7º - Aplicar uma cor para personalizar um pouco mais a barra de navegação com a classe .navbar-dark para letras claras ou .navbar-light para letras escuras.

Experimente reduzir o tamanho da tela.


Mostrar Código

<nav class="navbar navbar-expand-md bg-dark navbar-dark>
<a class="navbar-brand" href="#">LOGO</a>
<button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#id-ul">
<span class="navbar-toggler-icon"></span>
<button>
<div class="collapse navbar-collapse" id="id-div">
<ul class="navbar-nav">
<li class="nav-item">
<a class="nav-link" href="#">Link1</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link2</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link3</a>
</li>
</ul>
</div>
</nav>

Formulário em barras de navegação

Para inserir um formulário em uma barra de navegação precisamos apenas criar um elemento <form> dentro do elemento <nav>. O elemento <form> precisa ter a classe .form-inline para os elementos do formulário ficarem em linha.

Mostrar Código

<nav class="navbar navbar-expand btn-danger">
<ul class="navbar-nav">
<li class="nav-item">
<a class="nav-link" href="#">Link1</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link2</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link3</a>
</li>
</ul>
<form class="form-inline" action="/action_page.php">
<input class="form-control mr-sm-2" type="text" placeholder="Search"
<button class="btn btn-success" type="submit">Pesquisar...</button>
</form>
</nav>

.fixed-top .fixed-bottom e .sticky-top: Fixando barras de navegação

Podemos fixar a barra de navegação no topo e mesmo que a página seja rolada a barra de navegação irá sempre ficar no topo da página. Para isso precismos apenas inserir a classe .fixed-top na tag inicial do elemento <nav>. Experimente rolar o iframe.

Mostrar Código

<nav class="navbar navbar-expand-sm bg-darkviolet navbar-dark fixed-top">
<a class="navbar-brand" href="#">LOGO</a>
<ul class="navbar-nav">
<li class="nav-item">
<a class="nav-link" href="#">Link1</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link2</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link3</a>
</li>
</ul>
</nav>
