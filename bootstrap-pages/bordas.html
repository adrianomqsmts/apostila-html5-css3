Bordas com Bootstrap 4

Para definir uma borda padrão com os valores 1px solid white precisamos apenas inserir no elemento html a classe .border. Para facilitar a visualização, foi omitido dos trechos de código as classes .d-inline-block p-5 e m-1 do elemento <span> abaixo que os permitem ficar quadrados e um ao lado do outro, para ver como funciona essas classes acesse display e espaçamentos.

Ver código

<span class="border"></span>

Podemos definir Em quam parte do elemento a borda fique ausente através das classes .border-bottom-0, .border-top-0, .border-left-0 e .border-right-0, da mesma forma que podemos definir qual lado queremos definir um borda através das classes .border-bottom, .border-top, .border-left e .border-right.

Ver código

<span class="border border-bottom-0">.border-bottom-0</span>
<span class="border border-top-0">.border-top-0</span>
<span class="border border-left-0">.border-left-0</span>
<span class="border border-right-0">.borde-right-0</span>
<span class="border-bottom">.border-bottom</span>
<span class="border-top">.border-top</span>
<span class="border-right">.border-right</span>
<span class="border-left">.border-left</span>
.border-bottom-0 .border-top-0 .border-left-0 .border-right-0


.border-bottom .border-top .border-right .border-left

Bordas coloridas

Podemos colorir as bordas acrescentando as classes .border-primary, .border-secondary, .border-success, .border-danger, .border-warning, .border-info, .border-light, .border-dark ou .border-white. Lembrando que deve-se declarar sempre a classe .border ou as suas variações vistas acima.

Ver código

<span class="border border-primary">.border-primary</span>
<span class="border border-secondary">.border-secondary</span>
<span class="border border-success">.border-success</span>
<span class="border border-danger">.border-danger</span>
<span class="border border-warning">.border-warning</span>
<span class="border border-info">.border-info</span>
<span class="border border-light">.border-light</span>
<span class="border border-dark">.border-dark</span>
<span class="border border-white">.border-white</span>
.border-primary .border-secondary .border-success .border-danger
.border-warning .border-info .border-light .border-dark .border-white

Bordas Arredondadas

Podemos inserir arredondamento das bordas com as classes rounded e também podemos especificar onde arredondar através das variações .rounded-bottom, .rounded-top, .rounded-left e .rounded-right. Além disso podemos arredondar o elemento html como um circulo através da classe .rounded-circle.

Podemos ainda especificar em qual tamanho da janela de visualização teremos ou não arredondamento através dos tamanhos das telas com os valores sm, md e lg. Experimente reduzir o tamanho da janela do navegador.

Ver código

<span class="border rounded">.rounder</span>
<span class="border rounded-bottom">.rounded-bottom</span>
<span class="border rounded-left">.rounded-left</span>
<span class="border rounder-right">.rounder-right</span>
<span class="border rounded-circle">.rounded-circle</span>
<span class="border rounded-sm">.rounded-sm</span>
<span class="border rounded-md"> rounded-md</span>
<span class="border rounded-lg">.rounded-lg</span>
.rounded .rounded-bottom .rounded-left .rounder-right
.rounded-circle .rounded-sm .rounded-md .rounded-lg
