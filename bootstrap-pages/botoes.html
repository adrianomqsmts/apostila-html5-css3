Botões com Bootstrap 4

Para aplicar o estilo padrão de um botão B4 devemos usar a classes .btn, e podemos usar as classes .btn-*, em conjunto, para aplicar as cores padrões do B4 nos botões.

Classe Resultado
<button type="button" class="btn"Botão</button>
.btn .btn-primary
.btn .btn-secondary
.btn .btn-success
.btn .btn-info
.btn .btn-warning
.btn .btn-danger
.btn .btn-dark
.btn .btn-light
.btn .btn-link



btn-outline-*

Também exitem as classes de botões sem preenchimento com bordas coloridas (.btn-outline-*).

Classe Resultado
.btn .btn-outline-primary
.btn .btn-outline-secondary
.btn .btn-outline-success
.btn .btn-outline-info
.btn .btn-outline-warning
.btn .btn-outline-danger
.btn .btn-outline-light .text-dark
.btn .btn-outline-dark


Transformando Links e inputs

Podemos aplicar esse mesmo efeito nos elementos <input type="*"> e nos links <a href="*">.

Classe Resultado
<a href="" class="btn btn-outline-success">Botão</a> Botão
<input type="text" class="btn btn-warning">
<button type="button" class="btn btn-danger"Botão</button


Tamanho dos Botões em Bootstrap 4

Podemos aumentar ou diminuir o tamanho dos botões com classes B4.

Classe Resultado
.btn-lg Botão Grande
.btn-sm Botão Pequeno
.btn-block Botão 100% de largura
