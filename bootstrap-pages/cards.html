Cards com Bootstrap 4

Cards são usados para agregar uma informação e normalmente pode conter cabeçalhos, rodapés, cores, botões, etc.

.card e .card-body

Para criar um card básico devemos usar uma div com a classe .card e uma div filho com a classe .card-body

Mostrar Código

<div class="card">
<div class="card-body">Cartão básico</div>
</div>
Cartão básico


.card-header e .card-footer

As classes .card-header e .card-footer adicionam, respectivamente, um cabeçalho e um rodapé em um cartão.

Mostrar Código

<div class="card">
<div class="card-header">Cabeçalho</div>
<div class="card-body">Conteúdo</div>
<div class="card-footer">Rodapé</div>
</div>

.card-title .card-text e card-link

Podemos adicionar links dentro de um cartão usando a classe card-link, textos usando a classe .card-text e um título com a classe .card-title dentro de um elemento com a classe .card-body.

.stretched-link

Para transformar o cartão inteiro em clicável como se fosse um link, devemos adicionar a classe .stretched-link dentro do link, e então o cartão se comportará como um link. Para que o corpo de um cartão fique sobre a imagem precisamos adicionar a classe .card-img-overlay no elemento com a classe .card-body.

Mostrar Código

<div class="card">
<div class="card-body">
<h1 class="card-header">Título</h1>
<p class="card-text">Lorem ipsum dolor sit amet consectetur adipisicing elit. Id quos animi inventore dolore, eligendi ipsam commodi minima iste illum maxime minus et ratione explicabo doloribus in, esse, debitis iure laboriosam!</p>
<a href="" class="card-link stretched-link">LINK!</a>
</div>
</div>

.card-columns

A classe .card-columns nos permite criar uma organização que agrupa diversos cartões, onde o layout se ajusta automaticamente quando são inseridos novos cartões. Importante observar, que em janelas de visualizações menores, os cartões são exibidos verticalmente.

Mostrar Código

<div class="card-columns">
<div class="card bg-danger">
<div class="card-body">
<p class="card-text text-center">Cartão 1</p>
</div>
</div>
<div class="card bg-info">
<div class="card-body">
<p class="card-text text-center">Cartão 2</p>
</div>
</div>
<div class="card bg-success">
<div class="card-body">
<p class="card-text text-center">Cartão 3</p>
</div>
</div>
<div class="card badge-primary">
<div class="card-body">
<p class="card-text text-center">Cartão 4</p>
</div>
</div>
<div class="card bg-secondary">
<div class="card-body">
<p class="card-text text-center">Cartão 5</p>
</div>
</div>
<div class="card bg-warning">
