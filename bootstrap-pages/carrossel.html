Carrossel com Bootstrap 4

Um Carrossel permiti criar um conteúdo como slideshow, onde os elementos dicam em transição entre eles.

  1. Precisamos definir o container para o carrossel através da classe .carousel e um identificador. Em seguida, precisamos especificar o atributo data-ride="carousel" para especificar que o carrousel irá se mover.
    • <div id="slide" class="carousel" data-ride="carousel"></div>
  2. Criar um container para os indicadores que irão mostrar a quantidade e a posição atual do slide. Para definir um indicadores precisamos criar um lista, onde o elemento <ul> terá cada a classe .carousel-indicators.
    • <ul class="carousel-indicators"><ul>
    • Cada item da lista deve ter o atributo data-target="#slide" apontando para o identificador slide do container antes declarado. Além do atributo data-slide-to="0" que aponta para o número de qual slide queremos nos mover ao clicar no indicador. Por fim, devemos marcar o indicador inicial com a classe active mostrando que é o primeiro indicador ativo.
      • <ul class="carousel-indicators">
        <li data-target="#slide" data-slide-to="0" class="active"></li>
        <li data-target="#slide" data-slide-to="1"></li>
        <li data-target="#slide" data-slide-to="2"></li>
        </ul>
  3. Criar os itens dos slides. Precisamos primeiro criar um container com a classe .carousel-inner que irá conter cada item que conterá uma imagem. Cada item é um container com a classe .carousel-item que conterá, cada um deles, uma única imagem <img>. Devemos usar a classe active para especificar a imagem inicial.
    • <div class="carousel-inner">
      <div class="carousel-item active">
      <img src="../imagens/code01.jpg" alt="" style="width: 100%; height: 500px;">
      </div>
      <div class="carousel-item">
      <img src="../imagens/code02.jpg" alt="" style="width: 100%; height: 500px;">
      </div>
      <div class="carousel-item">
      <img src="../imagens/code03.jpg" alt="" style="width: 100%; height: 500px;">
      </div>
      </div>
  4. Criar os ícones de anterior e próximo. Para isso precisamos criar dois links, um com a classe .carousel-control-prev e outro com a classe .carousel-control-next, e ambos com os atributos href="#slide" apontando para o identificador do slide criado no passo 1.
    • Devemos ainda criar o ícone de próximo e anterior, para isso precisamos criar um elemento <span> dentro de cada link, e cada elemento <span> deverá ter a classe .carousel-control-prev-icon ou .carousel-control-next-icon referente ao seu link.
      • <a class="carousel-control-prev" href="#slide" data-slide="prev">
        <span class="carousel-control-prev-icon"></span>
        </a>
        <a class="carousel-control-next" href="#slide" data-slide="next">
        <span class="carousel-control-next-icon"></span>
        </a>
  5. (Opcional) Dar títulos e uma descrição para cada slide. Precisaremos definir dentro de cada item do slide, abaixo das imagens, um novo container com a classe .carousel-caption e dentro dele declarar um título como um header e um descrição como parágráfo.
    • <div class="carousel-inner">
      <div class="carousel-item active">
      <img src="../imagens/code01.jpg" alt="" style="width: 100%; height: 500px;">
      <div class="carousel-caption">
      <h1>HTML5</h1>
      <p>Venha Ver a nossa apostila de HTML5</p>
      </div>
      </div>
      <div class="carousel-item">
      <img src="../imagens/code02.jpg" alt="" style="width: 100%; height: 500px;">
      <div class="carousel-caption">
      <h1>CSS3</h1>
      <p>Venha Ver a nossa apostila de CSS3</p>
      </div>
      </div>
      <div class="carousel-item">
      <img src="../imagens/code03.jpg" alt="" style="width: 100%; height: 500px;">
      <div class="carousel-caption">
      <h1>Bootstrap 4</h1>
