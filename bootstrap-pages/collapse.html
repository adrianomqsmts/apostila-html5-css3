Collapse com Bootstrap 4

A classe .collapse são úteis para ocultar ou mostrar conteúdos.

Inicialmente, para abrir ou ocultar um conteúdo ou elemento HTML, precisamos definir um elemento clicável como um link ou botão.

Assim a classe .collapse dentro do elemento <div> indica que o elemento é um elemento expansível, ou seja, será o elemento que será ocultado ou mostrado ao se clicar no elemento clicável. Então, temos o elemento que pode ser tanto <a> quando <button> com o atributo data-toggle="collapse" indicando que este elemento será o responsável pela ação de ocultar ou desocultar um elemento, elemento este, que deve ter o seu identificador #id, único, especificado no atributo data-target="#id" de um botão ou no atributo href="#id" de um link.

Mostrar Código

<a href="#id" data-toggle="collapse">Link</a>
<div id="id" class="collapse">
Lorem ipsum dolor sit amet...
</div>
