Cores em textos

Existem um conjunto finito de classes que adicionam cores pré-definidas aos elementos. Essas classes também podem carregar um significado, se usada dentro do contexto.

Classe Resultado
.text-muted

Este texto está mutado.
.text-primary

Este texto é o mais importante.
.text-secondary

Este texto é menos importante.
.text-success

Este texto indica sucesso.
.text-info

Este texto indica uma informação.
.text-warning

Este texto indica um aviso.
.text-danger

Este texto indica um perigo.
.text-dark

Este texto é cinza escuro.
.text-white

Este texto é branco.
.text-body

Este texto é preto padrão
.text-light

Este texto é um cinza bem claro
.text-white-50

Este texto é um cinza bem claro com opacidade
.text-black-50

Este texto é preto com opacidade
