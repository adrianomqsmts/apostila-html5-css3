Display com Bootstrap 4

Para transformar um elemento em um elemento de bloco precisamos especificar a classe d-block

Ver código

<div class="bg-info text-dark">
<span class="d-block border p-2 m-1">.d-block</span>
<span class="d-block border p-2 m-1">.d-block</span>
<span class="d-block border p-2 m-1">.d-block</span>
</div>
.d-block .d-block .d-block

<div class="bg-info text-dark">

<span class="d-inline border p-2 m-1">.d-inline</span>
<span class="d-inline border p-2 m-1">.d-inline</span>
</div>

Para transformar um elemento em um elemento de bloco de linha precisamos especificar a classe d-inline-block

<div class="bg-info text-dark">

Display Responsivos

Podemos usar os valores sm, md, lg e xl em combinação com as classes vitas acima, para dependendo do tamanho da janela de navegação dos elementos se comportem de forma esperada. Reduza o tamanho da janela de navegação para ver o efeito.

<div class="bg-info text-dark">
<span class="d-md-inline-block d-sm-none border p-2 m-1">.d-md-inline-block d-sm-none</span>
<span class="d-md-inline-block d-block border p-2 m-1">.d-md-inline-block d-block</span>
<span class="d-xl-inline-block d-block border p-2 m-1">.d-md-inline-block d-block</span>
</div>
.d-md-inline-block d-none .d-md-inline-block d-block .d-xl-inline-block d-block
