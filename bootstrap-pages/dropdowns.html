Dropdown (lista suspensa) com Bootstrap 4

Para criar uma lista suspensa, precismos de um conjunto de classes B4.

  • .dropdown: Usado em um elemento para indicar uma lista suspensa.
  • .dropdown-toggle: Usado para indicar o botão será o elemento que esconde e abre a lista suspensa. Deve ser usado em conjunto com o atributo data-toggle="dropdown" dentro do elemento para indicar onde abrir a lista suspensa.
  • .dropdown-menu: Classe usada para indicar que a lista comum irá ser uma lista suspensa.
  • .dropdown-item: Indica cada um dos itens da lista que ficará dentro da lista suspensa.

Mostrar Código.

<div class="dropdown">
<button type="button" class="btn btn-primary dropdown-toggle" data-toggle="dropdown">

Dropdown

</button>
<div class="dropdown-menu">
<a class="dropdown-item" href="#">Link 1</a>
<a class="dropdown-item" href="#">Link 2</a>
<a class="dropdown-item" href="#">Link 3</a>
</div>
</div>

.active e .disabled

Podemos permitir que um link dentro de uma lista suspensa seja ativo ou desabilitado respectivamente com as classes .active e .disabled.

Mostrar Código

<div class="dropdown">
<button type="button" class="btn btn-primary dropdown-toggle" data-toggle="dropdown">

Dropdown

</button>
<div class="dropdown-menu">
<a class="dropdown-item active" href="#">Link 1</a>
<a class="dropdown-item disabled" href="#">Link 2</a>
<a class="dropdown-item active" href="#">Link 3</a>
</div>
</div>

.dropright .dropleft e .dropup

A classe .dropright define uma lista suspensa para o lado direito, e não para baixo como padrão. Também temos a classe .dropleft que define uma lista para o lado esquerdo e a classe .dropup que abre a lista suspensa para cima

Mostrar Código

<div class="dropdown dropright">
<button type="button" class="btn btn-success dropdown-toggle" data-toggle="dropdown">

Dropdown Right

</button>
<div class="dropdown-menu">
<a class="dropdown-item" href="#">Link 1</a>
<a class="dropdown-item" href="#">Link 2</a>
<a class="dropdown-item" href="#">Link 3</a>
</div>
</div>
