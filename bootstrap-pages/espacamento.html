Espaçamentos com Bootstrap 4

Padding

Para inserir um espaçamento interno em um elemento html precisamos especificar a classe p-1 que vai até o valor p-5 além do valor .p-auto. Para remover espaçamento interno usamos a classe p-0 ou suas variações p-*-0

Ver código

<span class="border p-1">.p-1</span>
<span class="border p-2">.p-2</span>
<span class="border p-3">.p-3</span>
<span class="border p-4">.p-4</span>
<span class="border p-5">.p-5</span>
<span class="border p-auto">.p-auto</span>
.p-1 .p-2 .p-3 .p-4 .p-5 .p-auto

Podemos ainda permitir um espaçamento interno automático com a classe p-auto ou responsivo com as classes p-sm-*,p-md-*,p-lg-* e p-xl-*. Experimente reduzir o tamanho da tela

Ver código

<span class="d-inline-block border p-1 p-sm-5">.p-1 p-sm-5</span>
<span class="d-inline-block border p-2 p-sm-3 p-md-5 p-lg-1">.p-2 p-sm-3 p-md-5 p-lg-1</span>
<span class="d-inline-block border p-xl-5">.p-xl-5</span>
<span class="d-inline-block border p-4 p-sm-2">.p-4 p-sm-2</span>
<span class="d-inline-block border p-sm-2">.p-sm-2</span>
.p-1 p-sm-5
.p-2 p-sm-3 p-md-5 p-lg-1
.p-xl-5
.p-4 p-sm-2
.p-sm-2

Podemos também especificar em qual direção queremos aplicar o espaçamento. Podendo ser um espaçamento interno na parte superior com pt-*-*, inferior com pb-*-*, na direita com pr-*-* ou então na esquerda com pl-*-*.

Ver código

<span class="d-inline-block border pt-5">.pt-5</span>
<span class="d-inline-block border pb-5">.pb-5</span>
<span class="d-inline-block border pl-5">.pl-5</span>
<span class="d-inline-block border pr-5">.pr-5</span>
<span class="d-inline-block border pt-sm-5 pb-md-5 pl-lg-5 pr-xl-5">.pt-sm-5 pb-md-5 pl-lg-5 pr-xl-5</span>
.pt-5
.pb-5
.pl-5
.pr-5


.pt-sm-5 pb-md-5 pl-lg-5 pr-xl-5 pt-sm-5 pb-md-5 pl-lg-5 pr-xl-5


Margins

Todas as variações que fizemos usando o padding podemos usar usando o margin com as classes m-*-* para criar um espaçamento externo ao elemento. Para remover espaçamento externo usamos a classe m-*-0

.m-1
.m-2
.m-3
.m-4
.m-5
.m-auto

.mt-5
.mb-5
.ml-5
.mr-5


.mt-sm-5 mb-md-5 ml-lg-5 mr-xl-5 mt-sm-5 mb-md-5 ml-lg-5 mr-xl-5
