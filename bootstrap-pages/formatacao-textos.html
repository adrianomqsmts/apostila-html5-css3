>Configuração Padrão

O Bootstrap usa por padrão um tamanho de fonte (font-size) igual a 16px com um espaçamento entre as linhas (line-height) de 1.5.

A família de fontes usada são "Helvetica Neue", Helvetica, Arial, sans-serif.

Existe um espaçamento menor para os parágrafos, onde a margem superior é 0, enquanto a margem inferir é equivalente a 16px.

>Cabeçalhos

A formatação dos cabeçalho do padrão de um HTML muda em questão de tamanho, mas nenhum alteração que ressalva alguma comparação. Entretanto, existem classes que normalmente são muito úteis para se aplicar em cabeçalhos os permitindo um maior variedade.

A classe .display-1 - .display-4 são usada para destacar ainda mais um título aumentado suas fontes (font-size) e um espessura menos destaca (font-weight).
