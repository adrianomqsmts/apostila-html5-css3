Formulários com Bootstrap 4

Formulários com bootstrap 4 possuem uma formatação global que aplicam pequenos estilos em cima do padrão dos navegadores permitindo uma maior compatibilidade com os estilos. Abaixo poderemos ver a estrutura básica de um formulário, de onde iremos partir para as demais classes B4. Como pode ver no código, cada par <label> e <input> deve ser envolvido com a classe .form-group.

Mostrar Código

<form action="">
<div class="form-group">
<label for="email">E-mail</label>
<input type="email" class="form-control" id="email">
</div>
<div class="form-group">
<label for="password">Senha</label>
<input type="password" class="form-control" id="password">
</div>
<div class="checkbox">
<label><input type="checkbox"> Lembrar-me</label>
</div>
<button type="submit" class="btn btn-success"> Submit</button>
</form>


form-inline: Formulários em linha.

Podemos especificar que o formulário seja contido em uma linha, sem espaçamento entre os elementos, através da classe .form-inline que deve ser especificada dentro do elemento <form>

Mostrar Código

<form action="" class="form-inline">
<div class="form-group">
<label for="email">E-mail</label>
<input type="email" class="form-control" id="email">
</div>
<div class="form-group">
<label for="password">Senha</label>
<input type="password" class="form-control" id="password">
</div>
<div class="checkbox">
<label><input type="checkbox"> Lembrar-me</label>
</div>
<button type="submit" class="btn btn-success"> Submit</button>
</form>



Tamanhos

Podemos definir a altura dos campos de entrada dos formulários usando as classes .form-control-lg para campos grande e .form-control-sm para campos menores.

Mostrar Código

<div class="form-group">
<label for="email">E-mail</label>
<input type="email" class="form-control form-control-lg" id="email">
</div>
<div class="form-group">
<label for="email">E-mail</label>
<input type="email" class="form-control form-control-lg" id="email">
</div>


Formulários na horizontal (Usando Grids)

Podemos especificar um formulário horizontal, onde cada rótulo e seus respectivo campo de entrada ficam alinhados na mesma linha, e em caso de janela pequenas, os elementos irão se portar verticalmente. Para definir formulários assim, precisamos usar colunas e alinhar os rótulo e os campos em cada linha do grid.

O primeiro passo é criar uma linha, no caso dos formulários, com a classe .form-row e para cada elemento com a classe .form-group especificar a quantidade de colunas que este elemento irá ocupar através das classes .col-*

Mostrar Código

<form action="">
<div class="form-row">
<div class="form-group col-md-6">
<label for="email">E-mail</label>
<input type="email" class="form-control" id="email">
</div>
<div class="form-group col-md-6">
<label for="password">Senha</label>
<input type="password" class="form-control" id="password">
</div>
</div>
<div class="form-row">
<div class="checkbox">
<label><input type="checkbox"> Lembrar-me</label>
</div>
<div class="form-group col-md-6">
<label for="nome">nome</label>
<input type="nome" class="form-control" id="nome">
</div>
</div>
<button type="submit" class="btn btn-success"> Submit</button>
</form>

Outra forma de alinhas os elementos do formulário em linha é definindo a classe .row em cada grupo .form-group, além de especificar a quantidade de colunas que cada elemento irá ocupar através da classe .col-*. Também precisamos especificar em cada label a classe .col-form-label-*

Mostrar Código

<form action="">
<div class="form-group row">
<label for="email" class="col-sm-2 col-form-label">E-mail</label>
<input type="email" class="form-control col-sm-10" id="email">
</div>
<div class="form-group row">
<label for="password" class="col-sm-2 col-form-label">Senha</label>
<input type="password" class="form-control col-sm-10" id="password">
</div>
<div class="form-group row">
<label for="nome" class="col-sm-2 col-form-label">Nome: </label>
<input type="nome" class="form-control col-sm-10" id="nome">
</div>
<button type="submit" class="btn btn-success"> Submit</button>
</form>


.form-check: checkbox e radio em Bootstrap 4

Assim como a classe .form-group agrupa os rótulos e os campos de entradas temos a classe .form-check para agrupar elementos de rótulo com os seus respectivos campos de seleção. Para deixar cada um dos parres lado a lada, diferente do padrão que é um par abaixo do outro, precisamos especificar a classe .form-check-inline. Também é necessário que especifiquemos em cada <label> a classe .form-check-label e em cada elemento <input> a classe .form-check-input






Input Groups

A classe .input-group cria um container para inserir algum conteúdo, como ícone ou texto, dentro do campo de entrada. O primeiro passo, depois de criar o container com a classe .input-group é criar um um elemento <span> com a classe .input-group-text para inserir o texto na parte da frontal do campo de entrada ou .input-group-append para inserir o conteúdo na direita do campo de entrada.

Mostrar Código

<form>
<div class="input-group mb-3">
<div class="input-group-prepend">
<span class="input-group-text">@ </span>
</div>
<input type="text" class="form-control" placeholder="Username">
</div>
<div class="input-group mb-3">
<input type="text" class="form-control" placeholder="Your Email">
<div class="input-group-append">
<span class="input-group-text">@example.com </span>
</div>
</div>
</form>
@
@example.com

Podemos também inserir qualquer elemento na esquerda de um campo de entrada com a classe .input-group-prepend e na parte da direita com o elemento .input-group-append

Mostrar Código

<form>
<div class="input-group mb-3">
<div class="input-group-append">
<button class="btn btn-success" type="button">OK</button>
</div>
<input type="text" class="form-control">
</div>
<div class="input-group mb-3">
<input type="text" class="form-control">
<div class="input-group-prepend">
<button class="btn btn-success" type="button">OK</button>
</div>
</div>
</form>
