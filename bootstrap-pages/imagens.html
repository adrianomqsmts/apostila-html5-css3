Imagens com Bootstrap 4

Podemos aplicar três diferentes estilos padrões para um imagem. A primeira é a classe .rounded que adiciona cantos arredondados na imagem.



A segunda classe .rounded-circle deixa a imagem arredondada.



A terceira classe .img-thumbnail transforma a imagem em uma miniatura com bordas.



Flutuar imagem com Bootstrap 4 (.float-*)

Podemos flutuar uma imagem em para à esquerda basta adicionar a classe .float-left, enquanto para flutuar uma imagem à direita usamos a classe .float-right

Centralizar a Imagem com Bootstrap 4

Para centralizar uma imagem usamos as classes .mx-auto (margin:auto) .d-block (display:block) dentro de um contâiner.



Imagem Responsiva com Bootstrap 4

Para permitir que a imagem se ajuste ao tamanho da janela de visualização basta adcionar a classe .img-fluid no elemento <img> para que a imagem se redimensione de forma adequada. A classe .img-fluid usa os valores max-width: 100%; e height: auto; para a imagem. Experimente reduzir o tamanho da janela.
