Menus de Navegação (navs e navbar) com Bootstrap 4

Para criar um simples menu de navegação precisamos primeiro definir uma lista <ul> com a classe .nav, seguida pelos itens <li>, cada um deles, com uma classe .nav-item, e por fim cada link <a> definido com uma classe .nav-link.

Podemos usar os links <a> em conjunto com as classes .active para definir o link atual ou a classe .disabled para desabilitar algum link.

Mostrar Código

<ul class="nav">
<li class="nav-item">
<a class="nav-link" href="#">Link1</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link2</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link3</a>
</li>
</ul>

Menus com Dropdown

Podemos usar os conhecimentos anteriores e juntá-los em um menu ainda mais único. Para isso, basta seguir a criação padrão de menu de navegação e em seguida, no item, introduza um link/botão dropdown que já conhecemos nos capítulos anteriores. Lembrando de especificar no elemento <li> a classe .dropdown.

Mostrar Código

<ul class="nav nav-tabs nav-justified">
<li class="nav-item">
<a class="nav-link" href="#">Link1</a>
</li>
<li class="nav-item dropdown">
<a class="nav-link dropdown-toggle" data-toggle="dropdown" href="#">Link2</a>
<div class="dropdown-menu">
<a class="dropdown-item" href="#">Link2.1</a>
<a class="dropdown-item" href="#">Link2.2</a>
<a class="dropdown-item" href="#">Link2.3</a>
</div>
</li>
<li class="nav-item">
<a class="nav-link" href="#">Link3</a>
</li>
</ul>

Menus Dinâmicos

Podemos criar um menu de navegação dinâmico, que ao clicar em um link, ao invés de abrir uma nova página, o elemento com o #id especificado é desocultado e os demais ocultados, dando a impressão que uma nova página carregou. Para isso precisamos apenas declarar o atributo data-toggle="tab ou pills" dentro dos links, especificando no atributo href="#id" o identificados da seção que deseja que o contúdo seja mostrado.

Também precisamos especificar inicialmente o elemento ativo, que inicialmente está visível através da classe .active e devemos também especificar os elementos que estarão inicialmente ocultos através da classe .fade.



Mostrar Código

<ul class="nav nav-pills nav-justified">
<li class="nav-item">
<a class="nav-link data-toggle="pill" active" href="#menu1">Link1</a>
</li>
<li class="nav-item">
<a class="nav-link" data-toggle="pill" href="#menu2">Link2</a>
</li>
<li class="nav-item">
<a class="nav-link" data-toggle="pill" href="#menu3">Link3</a>
</li>
</ul>

<div class="tab-content">
<div class="tab-pane container active" id="menu1">MENU 01</div>
<div class="tab-pane container fade" id="menu2">MENU 02</div>
<div class="tab-pane container fade" id="menu3">MENU 03</div>
</div>
