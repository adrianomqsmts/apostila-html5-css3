Paginação com Bootstrap 4

Para definir uma paginação simples B4 precismos de uma lista <ul> com a classe .pagination, com os itens <li> com a classe .page-item e os links <a> com a classe .page-link.

A classe .active é usada para destacar a página atual, enquanto a classe .disabled é usada para desabilitar alguma página e são usados no elemento <li>.

Mostrar Código

<ul class="pagination">
<li class="page-item">
<a href="" class="page-link"> Anterior </a>
</li>
<li class="page-item">
<a href="" class="page-link"> 1 </a>
</li>
<li class="page-item active">
<a href="" class="page-link"> 2 </a>
</li>
<li class="page-item">
<a href="" class="page-link"> 3 </a>
</li>
<li class="page-item">
<a href="" class="page-link"> 4 </a>
</li>
<li class="page-item disabled">
<a href="" class="page-link"> 5 </a>
</li>
<li class="page-item disabled">
<a href="" class="page-link"> Próximo </a>
</li>
</ul>

.pagination-*

Podemos definir uma paginação maior com a classe .pagination-lg ou uma paginação menor com a classe .pagination-sm em conjunto com a classe .pagination dentro do elemento <ul> : <ul class="pagination pagination-*">
