Spinners com Bootstrap 4

Para criarmos um spinner conhecido por carregamento, usamos a classe .spinner-border dentro de um elemento e o colorimos com as classes .text-*

Mostrar Código

<span class="spinner-border"></div>
<span class="spinner-border text-muted"></div>
<span class="spinner-border text-primary"></div>
<span class="spinner-border text-success"></div>
<span class="spinner-border text-info"></div>
<span class="spinner-border text-warning"></div>
<span class="spinner-border text-danger"></div>
<span class="spinner-border text-secondary"></div>
<span class="spinner-border text-dark"></div>
<span class="spinner-border text-light"></div>



.spinner-grow

A classe .spinner-grow define um formato diferente, que a classe anterior. Mas se comporta da mesma forma.

Mostrar Código

<span class="spinner-grow"></div>
<span class="spinner-grow text-muted"></div>
<span class="spinner-grow text-primary"></div>
<span class="spinner-grow text-success"></div>
<span class="spinner-grow text-info"></div>
<span class="spinner-grow text-warning"></div>
<span class="spinner-grow text-danger"></div>
<span class="spinner-grow text-secondary"></div>
<span class="spinner-grow text-dark"></div>
<span class="spinner-grow text-light"></div>




.spinner-*-sm

Podemos diminuir o tamanho dos spinner especificando a classe .spinner-border-sm ou .spinner-grow-sm em conjunto com a classe .spinner-border ou .spinner-grow

Mostrar Código

<span class="spinner-grow text-warning""></div>
<span class="spinner-grow spinner-grow-sm text-warning""></div>

Mostrar Código

<span class="spinner-border text-warning"></div>
<span class="spinner-border spinner-border-sm text-warning"></div>



Spinner Buttons

Podemos inserir o Spinner dentro de um botão, bastando seguir o modelo abaixo.

Mostrar Código

<button type="button" class="btn btn-success"
<span class="spinner-border"></span>
</button>


Mostrar Código

<button type="button" class="btn btn-success"
<span class="spinner-border spinner-border-sm"></span>
carregando...
</button>
