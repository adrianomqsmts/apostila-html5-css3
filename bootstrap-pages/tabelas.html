Tabelas com Bootstrap

Tabela Simples (.table)

A classe .table adiciona um estilo simples padrão do bootstrap à uma tabela com fundo transparente.

<table class="table">

Título 1 Título 2 Título 3
Dado 1.1 Dado 1.2 Dado 1.3
Dado 2.1 Dado 2.2 Dado 2.3
Dado 3.1 Dado 3.2 Dado 3.3
Dado 4.1 Dado 4.2 Dado 4.3


Tabela Listrada (.table-striped)

A classe .table com a classe .table-striped adiciona listras, em um tom mais cinza, mas linhas da tabela. Para melhor vermos, iremos inserir as classes .bg-white e .text-dark.

<table class="table table-striped bg-white text-dark">

Título 1 Título 2 Título 3
Dado 1.1 Dado 1.2 Dado 1.3
Dado 2.1 Dado 2.2 Dado 2.3
Dado 3.1 Dado 3.2 Dado 3.3
Dado 4.1 Dado 4.2 Dado 4.3


Tabela Bordada (.table-bordered)

Tabela Coloridas (.table-*)

Além de poder usar o background para dar uma cor de fundo, podemos usar as classes .table-primary .table-success .table-danger .table-info .table-warning .table-active .table-secondary .table-light .table-dark para colorir uma tabela padrão B4 com diversas cores. Além disso, podemos aplicar estas classes em cada <tr> de uma tabela para destacar alguma linha importante de acordo com a semântica de cor.

<tr class="table-primary">

<tr class="table-danger">

<tr class="table-warning">

<tr class="table-success">

Título 1 Título 2 Título 3
Dado 1.1 Dado 1.2 Dado 1.3
Dado 2.1 Dado 2.2 Dado 2.3
Dado 3.1 Dado 3.2 Dado 3.3
Dado 4.1 Dado 4.2 Dado 4.3

Podemos também colorir os títulos de uma tabela com as classes .thead-dark .thead-light

Título 1 Título 2 Título 3
Dado 1.1 Dado 1.2 Dado 1.3
Dado 2.1 Dado 2.2 Dado 2.3
Dado 3.1 Dado 3.2 Dado 3.3

Tabela Responsiva (.table-responsive)

Para criar um tabela responsiva devemos inserir a classe .table-responsive na tabela. Assim, caso o conteúdo exceda o tamanho da janela, será criada uma barra lateral dentro da tabela. Experimente reduzir o tamanho da janela.

Podemos especificar em qual tamanho de janela queremos que a tabela seja ajustada ao tamanho do conteúdo, sendo sm, xl, lg e md

<table class="table table-responsive-sm bg-white text-dark">

Título 1 Título 2 Título 3 Título 4 Título 5 Título 6 Título 7 Título 8 Título 9
