Flexbox

Flexbox é um novo modelo de layout além dos conhecidos anteriormente e mais moderno. Iremos aprender passo a passo como trabalhar com esse tipo de layout propriedade por propriedade.

Basicamente um flexbox trabalha com uma classe que normalmente é chamada de flex-container e que contém diversas divs denominadas de itens.

.flex-container{
display: flex;
backgound-color: green;
}
.flex-container > div{
backgound-color: white;
width: 50px;
margin: 10px;
text-align: center;
line-height: 3em;
font-size: 1.5em;
color: black;
}
<div class="flex-container">
<div>1</div>
<div>2</div>
<div>3</div>
<div>4</div>
</div>

flex-direction :

A propriedade flex-direction define em qual direção o contêiner deve empilhar os cada um dos itens flexíveis. Podendo ser:

  • bolumn: É o valor em que os itens flexíveis são empilhados verticalmente de cima para baixo.
  • column-reverse: O reverso da coluna. O valor empilha os itens flexíveis verticalmente, mas de baixo para cima.
  • row: O valorempilha os itens flexíveis horizontalmente da esquerda para a direita (padrão).
  • row-reverse: O reverso de linha valor empilha os itens flexíveis horizontalmente, mas da direita para a esquerda.
.flex-container{
display: flex;
flex-direction: column;
}

flex-wrap

A propriedade flex-wrap define se os itens devem "quebrar", ou seja, sair de suas posições originais para se ajustarem quando o tamanho da janela de visualização diminuir, especificando se os itens devem ficar abaixo dos outros quando o espaço for acabando.

  • wrap: O valor especifica que os flex items serão quebrados caso necessário.
  • nowrap: O valor especifica que os itens flex não irão quebrar (padrão).
  • wrap-reverse: O valor especifica que os itens flexíveis vão quebrar se necessário, mas na ordem inversa, ou seja, o primeiro item irá ser jogado para baixo, em seguida o segundo...
.flex-container{
display: flex;
flex-wrap: wrap;
}

justify-content

A propriedade justify-content é usada para alinhar os itens flexíveis, de acordo com cada linha, ou seja, na horizontal, das seguintes maneiras:

  • center: O valor alinha os itens flexíveis no centro do contêiner.
  • flex-start: O valor alinha os items flexíveis no início do contêiner (padrão).
  • flex-end O valor alinha os items flexíveis no final do contêiner.
  • space-around: O valor alinha os itens flexíveis com espaço antes, entre, e depois das linhas.
  • space-between: O valor alinha os itens flexíveis com espaço entre os linhas.
.flex-container{
display: flex;
height: 200px;
justify-content: center;
}

align-items

A propriedade align-items é usada para alinhar os itens flexíveis, de acordo com o tamanho do contêiner, da seguintes maneiras.

  • center: O valor alinha os itens flexíveis no centro do contêiner.
  • flex-start: O valor alinha os items flexíveis na parte superior do contêiner (padrão).
  • flex-end O valor alinha os items flexíveis na parte inferior do contêiner.
  • stretch: O valor estica os itens flexíveis para preencher o contêiner (padrão)
  • baseline: O valor alinha os itens flexíveis de acordo uma linha imaginária na base de todos os conteúdo dos itens. Ou seja, independetemente do conteúdo, nenhum conteúdo ficará abaixo da linha base, mesmo que os contêiners tenham tamanhos distintos.

order

A propriedade order especifica a ordem em que aparecem os itens flexíveis. Assim, podemos alterar a ordem em que os itens aprececem no código e no layout (Podendo evitar ter que reescrever ou reordenar os trechos de códigos). O valor especificado dever ser um número, sendo 0 o valor padrão.

<div class="flex-container">
<div style="order: 3">1</div>
<div style="order: 4">2</div>
<div style="order: 1">3</div>
<div style="order: 2">4</div>
</div>

flex-grow

A propriedade flex-grow especifica o quanto um item flexível deve crescer em relação aos demais itens flexíveis. O valor especificado deve ser um número, sendo 0 o valor padrão.

<div class="flex-container">
<div style="flex-grow: 0">1</div>
<div style="flex-grow: 0">2</div>
<div style="flex-grow: 0">3</div>
<div style="flex-grow: 8">4</div>
</div>

align-self

A propriedade align-self especifica o alinhamento para o item selecionado dentro do contêiner. Basicamente, a propriedade align-self substitui o alinhamento pela propriedade align-items, mas aplicada a um único item.

<div class="flex-container" style="height:200px;">
<div>1</div>
<div>2</div>
<div style="align-self: center">3</div>
<div style="align-self: flex-start">4</div>
</div>
