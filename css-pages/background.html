Background: fundo da tela

A propriedade de fundo (Background) é usada para definir os efeitos de fundo de um elemento HTML.

background-image: imagem do fundo

A propriedade background-image especifica uma imagem para ser usada como plano de fundo de um elemento HTML. Por padrão, caso a imagem tenha um tamanho menor que o elemento HTML, então a imagem ficara se repetindo, até que preencha todo o elemento html.

elemento { background-image: url(caminho/img); ...; }

Oie! Eu sou uma div com uma imagem de fundo
Clique aqui!
div {
background-image: url(imagens/img2.png);
height: 200px;
}


background-repeat: repetição do fundo

Caso a queiramos que a imagem não se repita ao longo na horizontal ou vertical, podemos usar a propriedade Background-repeat para especificar em qual sentido, queremos que a imagem se repita ou que ela não se repita para preencher o elemento HTML. Abaixo temos a imagem original:




Oie! Eu sou uma div com uma imagem de fundo
Clique aqui!
div {
height: 200px;
background-image: url(../imagens/img-3.png);
border: 1px solid green;
}

Oie! Eu sou uma div com uma imagem de fundo e com background-repeat: repeat-x;
Clique aqui!
div {
height: 200px;
background-image: url(../imagens/img-3.png);
border: 1px solid green;
background-repeat: repeat-x;
}

Oie! Eu sou uma div com uma imagem de fundo e com background-repeat: repeat-y;
Clique aqui!
div {
height: 200px;
background-image: url(../imagens/img-3.png);
border: 1px solid green;
background-repeat: repeat-y;
}

Oie! Eu sou uma div com uma imagem de fundo e com background-repeat: no-repeat;
Clique aqui!
div {
height: 200px;
background-image: url(../imagens/img-3.png);
border: 1px solid green;
background-repeat: no-repeat;
}


background-position: posicionamento do fundo

Ainda podemos definir o posicionamento da imagem em relação ao elemento HTML usando a propriedade background-position.

Código Resultado
div {
height: 200px;
background-image: url(imagens/img-3.png);
border: 1px solid green;
background-repeat: no-repeat;
}
#img-esquerda {
background-position: left;
}
Oi! Eu sou uma div com ID #img-esquerda
#img-direita {
background-position: right;
}
Oi! Eu sou uma div com ID #img-direita
#img-centro {
background-position: center;
}
Oi! Eu sou uma div com ID #img-centro
#img-direita-em-cima {
background-position: right top;
}
Oi! Eu sou uma div com ID #img-direita-em-cima
#img-esquerda-em-baixo {
background-position: left bottom;
}
Oi! Eu sou uma div com ID #img-esquerda-em-baixo
#img-centro-em-baixo {
background-position: bottom center;
}
Oi! Eu sou uma div com ID #img-centro-em-baixo


background-attach

A propriedade background-attach especifica se a imagem de fundo deve acompanhar ou não a barra de rolagem. Ou seja, a imagem quando rolamos a barra para baixo, deve subir ou ela deve continuar fixa na tela e continuar descendo?

Código Resultado
div {
height: 250px;
background-image: url(imagens/img-3.png);
border: 1px solid green;
background-repeat: no-repeat;
}
#img-scroll {
background-attachment: scroll;
}
Oi! Eu sou uma div com uma imagem que acompanha o scroll!
#img-fixed {
background-attachment: fixed;
}
Oi! Eu sou uma div com uma imagem fixada na tela, role a página.


Tudo de uma vez! Background

Agora que aprendemos todos as possibilidades de estilos possíveis usando a propriedade background, podemos aplicar todas essas propriedades em uma única linha usando a propriedade background.

É importante definir uma cor para o background, mesmo com uma imagem, pois, caso ocorra algum erro ou a imagem demore a ser carregada, ainda assim, a cor de fundo pode ser visualizada. A ordem dos valores que devem ser especificados são (alguns valores do background podem ser omitidos, assumindo valores padrões):

elemento {background: background-color background-image background-repeat background-attachment background-position; }

Oi! Eu sou uma div com a classe div-2 e com um background sem background-attachment e background-position!
Clique Aqui!
div {
height: 250px;
background: #875643 url(imagens/img2.png) no-repeat;
color: #000000;
}

Oi! Eu sou uma div com a classe div-2 e com um background sem background-attachment e background-position!
Clique Aqui!
div {
height: 250px;
background: #777777 url(imagens/img2.png) no-repeat fixed center center;
color: #000000;
}


Outas propriedades


  • A propriedade background-clip define a que distância o plano de fundo (cor ou imagem) deve se estender dentro de um elemento.
  • A propriedade background-origin especifica a posição de origem (o fundo área de posicionamento) de uma imagem de fundo.
  • A propriedade background-size especifica o tamanho das imagens de fundo.
