border: bordas

A propriedade border permite especificar o estilo, a largura e a cor da borda de um elemento.

border-style: estilo das bordas

A propriedade border-style especifica o tipo de borda exibir, podendo ter até quatro valores que difinem o tipo de parte da borda no sentido horário.

border-style: borda superior borda direita borda inferior borda esquerda
  • dotted: Define uma borda pontilhada
  • dashed: Define uma borda tracejada
  • solid: Define uma borda sólida
  • double: Define uma borda dupla
  • groove: Define uma borda com sulcos 3D.
  • ridge: Define uma borda 3D estriada.
  • inset: Define uma borda de inserção 3D.
  • outset: Define uma borda de início 3D.
  • none: Não define nenhuma fronteira
  • hidden: Define uma borda oculta
 div  {
    border-style: dotted;
}
Oi! Eu sou uma div com border-style: dotted 
 div  {
    border-style: dashed;
}
Oi! Eu sou uma div com border-style: dashed 
 div  {
    border-style: solid;
}
Oi! Eu sou uma div com border-style: solid 
 div  {
    border-style: double;
}
Oi! Eu sou uma div com border-style: double 
 div  {
    border-style: groove;
}
Oi! Eu sou uma div com border-style: groove 
 div  {
    border-style: ridge;
}
Oi! Eu sou uma div com border-style: ridge 
 div  {
    border-style: inset;
}
Oi! Eu sou uma div com border-style: inset 
 div  {
    border-style: none;
}
Oi! Eu sou uma div com border-style: none 
 div  {
    border-style: hidden;
}
Oi! Eu sou uma div com border-style: hidden 
 div  {
    border-style: dotted solid double inset;
}
Oi! Eu sou uma div com border-style: dotted solid double inset 
 div  {
    border-style: double none;
}
Oi! Eu sou uma div com border-style: double none

border-width: espessura das bordas

A propriedade border-width define a largura de todas as quatro bordas, usando valores específicos ou predefinidos como thin (fino), medium (médio) ou hick (espesso)

Também podemos especificar o a espessura de cada borda no sentido horário

border-width: borda superior borda direita borda inferior borda esquerda 
 div  {
    border-style: solid;
    border-width: 10px;
}
Oi! Eu sou uma div com border-style: solid e border-width: 10px 
 div  {
    border-style: double;
    border-width: medium;
}
Oi! Eu sou uma div com border-style: double e border-width: medium 
 div  {
    border-style: solid;
    border-width: 1px thick;
}
Oi! Eu sou uma div com border-style: solid e border-width: 1px thick. 
 div  {
    border-style: solid;
    border-width: 5px 10px 1px 30px;
}
Oi! Eu sou uma div com border-style: solid e border-width: 10px 20px 1px 30px.

border-color: Cores das bordas

A propriedade border-color é usada para definir a cor das quatro bordas de um elementos HTML.

border-color: borda superior borda direita borda inferior borda esquerda 

 div  {
    border-style: solid;
    border-width: 5px;
    border-color: red;
}
Oi! Eu sou uma div com border-style: solid , border-width: 5px e border-color: red. 
 div  {
    border-style: double;
    border-width: 20%;
    border-color: hsl (50%, 100%, 50%);
}
Oi! Eu sou uma div com border-style: solid , border-width: 5px e border-color: hsl(50, 70%, 30%). 
 div  {
    border-style: groove;
    border-width: 1em;
    border-color: rgb(50,50,200);
}
Oi! Eu sou uma div com border-style: groove , border-width: 1em e border-color: rgb(50,50,200) 
 div  {

Oi! Eu sou uma div com border-style: groove double, border-width: 1em 15px e border-color: rgb(50,50,200) #ababab 
 div  {
    border-style: solid;

    border-color: blue green red yellow;

Oi! Eu sou uma div com border-style: solid , border-width: 5px e border-color: blue green red yellow

Tudo de uma vez! Border

Agora que aprendemos todos as possibilidades de estilos possiveis usando a propriedade border, podemos aplicar todas essas propriedades em uma única linha.

A propriedade border é uma forma abreviada para as seguintes propriedades individuais de borda.

elemento { width style color } 

 div  {
    border: 5px solid blue;
    background-color: black;
}
Oi! Eu sou uma div com border: 5px solid blue e background-color: black.

border-top border-left border-right border-bottom

Também é possível trablhar apenas de um lado das bordas de forma totalmente independente usando as propriedades border-top, border-left, border-right e/ou border-bottom. Elas funcionam da mesma forma que uma propriedade border e possuem também, suas variações em cor, espessura e estilo.

  • border-bottom, border-bottom-color, border-bottom-style, border-bottom-width
  • border-left, border-left-color, border-left-style, border-left-width
  • border-right-color, border-right-style, border-right-width,
  • border-top, border-top-color, border-top-style, border-top-width
 div  {
    border-top: 5px solid blue;
}
Oi! Eu sou uma div com border-top: 5px solid blue 
 div  {
    border-top-style: double;
    border-bottom-style: inset;
    border-bottom-color: red;
}
Oi! Eu sou uma div com border-top-style: double ,border-bottom-style: inset e border-bottom-color: red

border-radius: arredondamento de bordas

A propriedade border-radius é usada para especificar o arredondamento de todas as bordas de um elemento HTML. Também é possivel aplicar o arredondamento em cada borda de forma individual.

  • border-top-left-radius, border-top-right-radius
  • border-bottom-left-radius, border-bottom-right-radius
 div  {
    border: 5px solid blue;
    border-radius: 5px;
}
Oi! Eu sou uma div com border: 5px solid blue e border-radius: 5px. 
 div  {
    border: 6px double lawngreen;
    border-radius: 5px;
}
Oi! Eu sou uma div com border: 6px double lawngreen e border-radius: 15px. 
 div  {
    border: 6px double mediumspringgreen;
    border-bottom-left-radius: 5px;
}
Oi! Eu sou uma div com border: 6px double mediumspringgreen e border-bottom-left-radius: 15px. 
 div  {
    border: 6px double turquoise;
    border-top-right-radius: 5px;
}
Oi! Eu sou uma div com border: 6px double turquoise e border-top-right-radius: 15px.
