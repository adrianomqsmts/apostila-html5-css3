box-sizing - tamanho da caixa

Você pode ver até aqui, que os elementos aumentam de tamanho na medidada em que aplicamos diveros tipos de espaçamentos deixando o elemento maior do que gostaríamos que ficasse. Para impedir esse crescimento podemos usar a propriedade box-sizing. 

     div  {
        border: 1px solid orange;
        width: 150px;
        height: 150px;
        margin: auto;
    }

Caixa SEM box-sizing e com espaçamento interno 
     div  {
        border: 1px solid orange;
        width: 150px;
        height: 150px;
        margin: auto;
        padding: 10px;
    }
Conteúdo
Caixa COM box-sizing e com espaçamento interno 
     div  {
        border: 1px solid orange;
        width: 150px;
        height: 150px;
        margin: auto;
        padding: 10px;
        box-sizing: border-box;
    }
