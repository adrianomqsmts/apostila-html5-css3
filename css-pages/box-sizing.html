box-sizing - tamanho da caixa

Você pode ver até aqui, que os elementos aumentam de tamanho na medida em que aplicamos diversos tipos de espaçamentos deixando o elemento maior do que gostaríamos que ficasse. Para impedir esse crescimento podemos usar a propriedade box-sizing.


Código Resultado
div {
border: 1px solid orange;
width: 150px;
height: 150px;
margin: auto;
}
Conteúdo
Caixa SEM box-sizing e com espaçamento interno
div {
border: 1px solid orange;
width: 150px;
height: 150px;
margin: auto;
padding: 10px;
}
Conteúdo
Caixa COM box-sizing e com espaçamento interno
div {
border: 1px solid orange;
width: 150px;
height: 150px;
margin: auto;
padding: 10px;
box-sizing: border-box;
}
