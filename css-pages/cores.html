Cores com CSS3

Para aplicarmos uma cor em elementos ou atributos em CSS podemos usar valores predefinidos em inglês como blue (azul), red (vermelho), green (verde), dentre outros 140 suportadas. Ou então usar valores RGB, HEX, HSL, RGBA ou HSLA.

Valores de Cores

Podemos definir a cor especificando o o valor numérico. Ao optar dessa forma, podemos tornar a cor mais personalizada para o estilo de uma página.

Cores RGB e RGBA

Em CSS uma cor pode ser específicada como valor RGB.


rgb ( vermelho, verde, azul )


Basicamente, cada parâmetro define a intensidade da cor com valores que variam entre 0 e 255, formando uma cor resultante da mistura das três cores. Por exemplo: Se usarmos rgb(255,0,0) a cor resultante é vermelha pura, pois não há mistura com as demais cores. O preto é a ausência de cor, enquanto o branco é a mistura de todas as cores. Os tons de cinza ocorrem quando os três valores são iguais.

Clique aqui para ver o código para a cor : rgb(0, 0, 0)

background-color: rgb(0,0,0);
}

Cores Hexadecimais

Funciona da mesma forma que o RGB, entretanto, em um formato hexadecimal.

#rrggbb

Onde rr (vermelho), gg(verde) e bb (azul) são valores hexadecimal entre 00 e ff (igual ao intervalo 0-255 em decimal). Por exemplo: O valor #ff0000 define uma cor vermelha, pois ff é o valor mais alto.

Clique aqui para ver o código para a cor : #000000

background-color: #000000;
}

Clique aqui para ver o código para a cor : #0000ff
