Layout - Display

A propriedade display é considerada uma das mais importantes para se controlar o layout das páginas. Basicamente a propriedade display especifica como um elemento é exibido. Normalmente um elemento tem por padrão a exibição block ou inline.



Layout inline - Elementos de linha

Os elementos inline não começam em uma nova linha e só ocupam a largura necessária para o conteúdo. Alguns exemplos são os elementos <span>, <a> e <img>

Código Resultado
span {
border: solid green;
}

Este é um elemento <span> inline e outro elemento <span> inline dentro de um parágrafo.



Layout block - Elementos de Níveis de Bloco

Um elemento de nível de bloco sempre começa em uma nova linha e ocupa toda a largura disponível. Os principais elementos de bloco são <div>, <p>,<h1>-<h6>.

Código Resultado
p {
border: solid green;
}

O parágrafo <p> é um elemento de nível de bloco

O parágrafo <p> é um elemento de nível de bloco

Ao usarmos então a propriedade display:block podemos transformar um elemento inline em um elemento block, e se usarmos display:inline podemos transformar um elemento block em inline. Ao usarmos display:none o elemento é ocultado, removendo todo o espaço que antes o elemento ocupava. Isso difere se usarmos a outra propriedade, a visibility: hidden; pois ela apenas oculta o objeto, entretanto o espaço ocupado pelo elemento continua.



Layout inline-block - Elementos blocos de linha

A propriedade display:inline-block basicamente mistura as qualidade de ambas. Um elemento bloco de linha permite que as bordas e margens sejam respeitadas não sobrepondo outros elementos (acontece com o inline), permite que o elemento tenha altura e largura definida (não acontece com o inline) e não gera uma quebra de linha entre os elementos.

Código Resultado
span {
display: inline;
border: solid green;
padding: 2px;
width: 100px;
}
