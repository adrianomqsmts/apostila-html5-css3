Especificidade

Quando temos duas regras CSS tentando aplicar um estilo em um mesmo elemento, normalmente a regra mais específica vence. Um exemplo seria, seletores #id vencem seletores .classe, que por sua vem vencem seletores comuns.

Abaixo temos uma lista de especificidade, onde o item mais alto na lista vence os demais para aplicar o mesmo estilo.

  1. Estilo Embutido (style="")
  2. Identificadores (#id)
  3. Classes (.class), atributos ([atributos]) e pseudo-classes (:hover)
  4. Elementos (h1,p,div..) e pseudo-elementos(::before)
Código Resultado
#titulo {
color: blue;
}
h1 {
color: red;
}

Eu sou um título com ID titulo
#titulo {
color: blue;
}
.titlo2 {
color: red;
}

Eu sou um título com ID titulo e classe titulo2
