Layout- Float and Clear

A propriedade float especifica como um elemento deve flutuar, basicamente usada para posicionar e formatar conteúdos.

Podemos fazer um elemento HTML flutuar na esquerda (left), na direita (right), sem flutuação (none padrão) ou herdar a flutuação do elemento pai (inherit).

Abaixo temos uma imagem com float:left dentro de um paragrafo p. A imagem se posiciona ao lado esquerdo do texto, e por sua vez o texto se posiciona ao lado direito da imagem.

img {
float: left;
padding-right: 1em;
}

Entretanto, ao usarmos a propriedade de flutuação em um elemento, os próximos elementos irão preencher o espaço flutuado e podemos não querer que isso aconteça.

.div1 {
float: left;
width: 20%;
height: 3em;
border: solid gold;
}

.div2 {
width: 30%;
border: solid blue;
}

.div3 {
width: 40%;
border: solid red;
}

div2

Para corrigir esse fluxo podemos usar a propriedade clear para quebrar o fluxo de flutuação dos elementos. Os possíveis valores para a propriedade clear são:

  • none - Os elementos podem flutuam em ambos os lados.
  • left - Nenhum elemento pode flutuar no lado esquerdo
  • right - Nenhum elemento pode flutuar no lado direito
  • both - Nenhum elemento pode flutuar em nenhum lado
  • inherit - O elemento herda a propriedade do pai
.div1 {
float: left;
width: 20%;
height: 3em;
border: solid gold;
}

.div2 {
clear: both;
width: 30%;
border: solid blue;
}

.div3 {
width: 40%;
border: solid red;
}

Caso ocorra de um elemento transbordar e o elemento estiver flutuando podemos corrigir isso, usando um tipo de hack clearfix.

div {
border: solid green;
height: 50px;
}

div img {
float: left;
}
Imagem transbordada e flutuado no lado esquerdo, dentro de uma div.

Um hack clearfix basicamente contém content: ""; clear: both; e display: table;

div {
border: solid green;
height: 50px;
content: "";
clear: both;
display: table;
}

div img {
float: left;
}
