Formas de Estilos

Existem 3 formas de aplicar estilos dentro de um documento HTML.

Estilo Inline

O estilo inline (em linha) é uma forma embutida que pode ser usada para aplicar um estilo único a um único elemento HTML.

Para usarmos a forma de estilo em linha, basta adicionarmos o atributo global style dentro da tag inicial de um elemento, seguida pelas propriedades CSS que desejamos aplicar.

OBS: No estilo em linha, caso o valor possua mais de uma palavra, deve-se usar aspas simples (exemplo 'Times New Roman').

<tagname style="propriedade1:valor1; ...;" >

<p style="text-align:left;color:yellow;">
Oi! Eu sou um parágrafo com o estilo em linha.
</p>

Oi! Eu sou um parágrafo com o estilo em linha.

Entretanto, ao aplicarmos o estilo inline, estamos perdendo as vantagem que as folha de estilo nos proporciona. Pois, ao misturar o conteúdo com a apresentação, estamos ainda assim, misturando conteúdo e apresentação. Então use o método inline com moderação.

Estilo Interno

Uma folha de estilo interna pode ser usada se uma única página HTML tiver um estilo exclusivo.

O estilo interno é definido dentro do elemento <style> dentro da cabeçalho da seção <head>.

<head>
<style>

p {
text-align: left;
color : yellow;
}
</style>
</head>
<body>
<p> Oi! Eu sou um parágrafo com estilo interno.</p>
</body>

Oi! Eu sou um parágrafo com estilo interno.

Estilo Externo

Com uma folha de estilos externa, podemos alterar o estilo de várias páginas ao mesmo tempo, alterando apenas um único arquivo externo com a extensão .css.

Cada página HTML deve incluir uma referência ao arquivo da folha de estilo externo dentro do elemento <link>, dentro da seção de cabeçalho <head>

<head>
<link rel="stylesheet" href="arquivo.css">
</head>

Um arquivo CSS deve ter a extensão .css, não pode conter nenhuma tag HTML e deve ter a seguinte aparência:

p {
text-align: left;
color : yellow;
}

h1 {
text-align: center;
color : blue;
}

Múltiplas Fromas de Estilos

Caso ocorra de utilizarmos vários arquivos externos e mais de um alterar o mesmo elemento HTML, então, o último arquivo lido terá preferência sobre os demais para alterar esse elemento.

Por exemplo: Dois arquivos alteram a cor de um parágrafo, então, o último arquivo inserido irá definir a cor que será aplicada no parágrafo.

Caso utilizarmos múltiplas formas de estilos ao mesmo tempo como inline, interna e externa, então os estilos serão "cascateados" de tal forma que o estilo inline tem preferência/prioridade sobre o estilo interno, que tem preferência/prioridade sobre o estilo externo.

INLINE > INTERNO > EXTERNO
