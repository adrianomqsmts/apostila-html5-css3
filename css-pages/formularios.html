Formulários

Aqui usaremos diferentes tipos de seletores para aplicar diversos estilos aos diferentes campos de um formulário.

input[type="text"] {
border: none;
border-bottom: 0.2em solid black;
padding: 0.5em;
width: 100%;
}
input[type="password"]:hover {
border: none;
border-bottom: 0.2em solid black;
padding: 0.5em;
width: 100%;
}
input[type="text"] {
border: none;
border-bottom: 0.2em solid black;
padding: 0.5em;
width: 100%;
}
input[type="text"]:focus {
background-color: blue;
}
input[type="search"] {
border: none;
border-bottom: 0.2em solid black;
padding: 0.5em;
width: 100%;
background-image: url(imagens/iconsearch.png);
background-repeat: no-repeat;
background-size: 2.5em;
padding-left: 2.5em;
}
textarea {
border: none;
border-bottom: 0.2em solid black;
width: 100%;
height: 300px;
text-align: justify;
resize: none;
}
