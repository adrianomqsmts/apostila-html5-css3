Gradientes

Podemos aplicar gradientes de cores em diferentes elementos HTML através da função linear-gradient ou radial-gradient.

Os parâmetros da função são:

background-image: linear-gradient(direção, cor1, cor2, ...);
background-image: radial-gradient(formato, cor1, cor2, ...);

Código Resultado
div {
background-image: linear-gradient(to bottom, red, yellow);
}
div {
background-image: linear-gradient(to right, red, yellow, green);
}
div {
background-image: linear-gradient(to right, rgba(255,0,0,0),rgba(0,2555,0,0.5), rgba(0,0,255,1));
}
div {
background-image: lradial-gradient(circle, red, yellow, green);;
}
