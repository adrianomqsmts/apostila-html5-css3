height and width: altura e largura

As propriedades width e height são usada para definir a largura e altura, respectivamente, dos elementos HTML. A altura e largura são para o conteúdo, não incluindo, bordas, espaçamentos internos ou externos. Para definir o valor dessas propriedades podem ser usados valores de unidades ou nominais como initial que redefine o tamanho padrão ou inherit que herdará o tamanho do elemento pai.

Código Resultado
div {
border: 1px solid orange;
width: 30px;
height: 30px;
}
Div 50x50px
div {
border: 1px solid orange;
width: 10cm;
height: 5cm;
}
Div 5x10 cm
div {
border: 1px solid orange;
width: 50%;
height: 50%;
}
Div 50x50%

Você pode ter notado que ao longo dos exemplos, que ao diminuir a tela, alguns elementos não acompanham, pois possuem valores absolutos e não relativos. Para que isso não ocorra em seus projetos, usem sempre valores relativos. Também podemos usar as propriedades max-width, max-height, min-height e min-width que especificam o tamanho máximo e mínimo que um elemento deve ter dependendo do tamanho de uma janela.

Código Resultado
div {
border: 1px solid orange;
max-width: 400px;
max-height: 400px;
}
Div 400x400px. Experimente diminuir tamanho da janela do navegador.
