INTRODUÇÃO

CSS (Cascading Style Sheets/ Folha de Estilos em Cascata) padronizada pela W3C, descreve como os elementos HTML devem ser exibidos na tela, paper ou em outra mídia. Com o objetivo de tratar das características de design da maneira mais efetiva e melhorar o gerenciamento das páginas de um site, foi criada uma extensão do HTML.

A principal vantagem de se usar CSS é a praticidade de economizar muito trabalho, podendo controlar o estilo (layout) de várias páginas web de uma única vez.

  • Espaçamento entre linhas;
  • Posicionamento de imagens e textos;
  • Cores e imagens de fundo;
  • Tratamento de fontes;
  • Margem e endentação;
  • Tratamento de características dos links;f
  • Dentre outras características de design.

Assim, o HTML fica livre apenas para descrever o conteúdo dos sites ficando mais limpo e objetivo, enquanto o CSS fica com o trabalho exclusivo de definir estilos para suas páginas web, como o design, layout, portabilidade, etc.

As principais vantagem de se usar CSS são:

  • Separar a definição do layout das páginas do conteúdo a ser exibido;
  • Padronização das páginas de um site;
  • Código HTML mais simples e fácil de ser manipulado;
  • Otimização para qualquer resolução de monitor de vídeo e
  • Controle mais preciso e previsível do formato da apresentação.


CSS Sintaxe

O seletor aponta para o elemento HTML que você deseja aplicar algum estilo (h1 no exemplo), enquanto o bloco de declaração contém uma ou mais declarações separadas por ponto e virgula.

seletor{bloco de declarações}
seletor{declaração1;declaração2;...;}

Cada declaração inclui um nome de propriedade CSS e um valor separados por dois pontos (color:red e font-size:12px no exemplo).

seletor{propriedade:valor}

Assim, todas as declarações são separadas por ponto-e-virgula e as declarações são cercadas por chaves. (todo o exemplo)

seletor{propriedade1:valor; propriedade2:valor2;...;}

Uma propriedade pode ter mais de um valor, nesse caso, deve-se separar cada valor por um único espaço.

seletor{propriedade1:valor1.1 valor1.2 valor1.3;}

Caso algum valor tenha mais de uma palavra, deve-se usar aspas duplas (por exemplo "Times New Roman").
