Links

Podemos aplicar difrentes efeitos para os links, desde, apenas remover o underline, até transformá-los em um botão ou menu.

Um link tem vários estados e pode assumir diferentes estilos de acordo com cada estado.

  • a:link : Um link não visitado
  • a:visited : Um link já acessado
  • a:hover: Ação quando o usuário passa o mouse sobre
  • a:active: Ação que ocorre quando o link é clicado.

Sempre que for aplicar estilos à um link. Deve-se sempre, aplicar primeiro o estilo para a:link, depois para a:visited, em seguida para a:hover e por último para a:active. 

    a:link {
        color: green;
    }
    
    a:visited {
        color: red;
    }
    
    a:hover {
        color: violet;
    }
    
    a:active {
        color: pink;
    }
