margins: espaçamento externo

A propriedade margin é usada para criar um espaçamento em torno dos elementos. Também é possível especificar a margem em cada lado de um elemento HTML. O valor auto centraliza o elemento (somente se o elemento tiver height e width atribuídos), enquanto o valor inherit herda a margem do elemento que contém o elemento com o valor (elemento pai).

  • margin
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
Código Resultado
div {

border: 1px solid white;
}
Oi! Eu sou uma div com border: 6px solid white e sem margens.
div {
margin: 30px;
border: 1px solid white;
}
Oi! Eu sou uma div com border: 6px solid white e com margin: 30px.
div {
margin: 70px;
border: 1px solid white;
}
Oi! Eu sou uma div com border: 6px solid white e com margin: 70px.
div {
margin-top: 10%;
border: 1px solid white;
}
Oi! Eu sou uma div com border: 6px solid white e com margin-top: 10%.
div {
margin-left: 5em;
border: 1px solid white;
}
Oi! Eu sou uma div com border: 6px solid white e com margin-left: 5em.
div {
margin: 10px 20px 30px 50px;
border: 1px solid white;
}
Oi! Eu sou uma div com border: 6px solid white e com margin: 10px 20px 30px 50px.
div {
margin: auto;
width: 30px;
height: 100%;
border: 1px solid white;
}
Oie! Estou centralizado.
