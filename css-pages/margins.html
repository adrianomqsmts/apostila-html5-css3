margins: espaçamento externo

A porpriedade margin é usada para criar um espaçamento em torno dos elementos. Também é possível especificar a margem em cada lado de um elemento HTML. O valor auto centraliza o elemento, enquanto o valor inherit herda a margem do elemento que contém o elemento com o valor (elemento pai).

  • margin
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
 div  {
    margin: 6px;
    border: 2px solid turquoise;
}
Oi! Eu sou uma div com border: 6px solid turquoise e com margin: 6px. 
 div  {
    margin: 30px;
    border: 2px solid turquoise;
}
Oi! Eu sou uma div com border: 6px solid turquoise e com margin: 30px. 
 div  {
    margin-top: 10%;
    border: 2px solid turquoise;
}
Oi! Eu sou uma div com border: 6px solid turquoise e com margin-top: 10%. 
 div  {
    margin-left: 5em;
    border: 2px solid turquoise;
}
Oi! Eu sou uma div com border: 6px solid turquoise e com margin-left: 5em. 
 div  {
    margin: 10px 20px 30px 50px;
    border: 2px solid turquoise;
}
Oi! Eu sou uma div com border: 6px solid turquoise e com margin: 10px 20px 30px 50px. 
 div  {
    margin: auto;
    width: 30px;
    height: 30px;
    border: 2px solid turquoise;
}
Oie! Estou centralizado.
