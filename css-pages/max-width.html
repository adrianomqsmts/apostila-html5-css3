Layout - max-width: Largura máxima

É sempre recomendado o uso da propriedade max-width pois ela define uma largura máxima que um elemento deve ter. Assim, ele pode assumir valores menores caso necessários, como em telas pequenas.

Outro detalhe importante. Para centralizar elementos é recomendado usar margin:auto. Assim, automaticamente o espaçamento de todos os lados serão ajustados para que o elemento fique centralizado.



div {
border: 20px solid green;;
width: 1000px;
margin: auto;
}
Oi! Eu sou um div com border: 20px solid green; width: 1000px e margin: auto;
