Media Queries

Com a regra @media, podemos definir diferentes regras css para um mesmo elemento HTML para diferentes tipos de mídias (telas) diferentes.

Assim, podemos definir um estilo personalizado para desktops, laptops, tablets e celulares.

@media not|only mediatype and (expressions) { CSS-Code; }

O resultado da consulta é verdadeiro se o tipo de midia especificado corresponde ao tipo de dispositivo que está visualizando o documento, se a consulta for verdadeira, então o código CSS é aplicado no documento.

O tipo de consulta mais comum é o tipo de telas de computadores (screen). No exemplo abaixo, estamos testando as telas dos dispositivos, se a tela tiver até 992px o background-color será azul, se a tela tiver no máximo 600px então a tela será amarela. Se a tela for maior que 992px então a div será verde. Experimente reduzindo o tamanho da tela.

div{
background-color: green;
}
@media screen and (max-width: 992px) {
div{
background-color: blue;
}
}
@media screen and (max-width: 600px) {
div{
background-color: yellow;
}
}
OIE

Podemos esconder elementos em telas menores

@media screen and (max-width: 600px) {
div{
display: none;
}
}
