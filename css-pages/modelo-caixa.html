Modelo de Caixa

Você pode ter notado até aqui, que os elementos HTML são formados por caixas retangulares. Essas caixas envolvem todos os elementos HTML e podem ser manipuladas de diversas formas. Abaixo podemos ver as propriedades que compõem um elemento HTML e que formam um modelo de caixa.

Importante notar então, que o tamanho do elemento HTML nesse ponto depende do tamanho que especificarmos

Assim a altura e largura de um elemento HTML, é a soma, das alturas e larguras de todas as propriedades CSS.

Nas próximas seções iremos ver como manipular cada uma das propriedades.
