Opacidade

A propriedade opacity especifica a opacidade ou transparência de um determinado elemento HTML tanto imagens quanto blocos. Como vimos em cores, isso também pode ser feito em alguns casos usando RGBA. O seu valor varia de 0.0 até 1.0

Código Resultado
img {
opacity: 0.2;
}
img {
opacity: 0.5;
}
img {
opacity: 0.7;
}
img {
opacity: 1.0;
}
img:hover {
opacity: 0.3;
}
