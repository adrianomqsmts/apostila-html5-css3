outline: contornos

Além do conteúdo, do espaçamento interno, da borda e do espaçamento externo, temos ainda o contorno do elemento HTML. Mas você pode ter notado, que no modelo de caixa não haviam contornos, e é exatamente isso. O contorno de um elemento não ocupa um espaço ou redefine o tamanho de um elemento.

Basicamente, o contorne é uma linha desenhada do lado de fora de uma borba, fazendo o elemento ter um estilo ainda mais único. Um contorno se assemelha a todas as possibilidades de uma borda, sendo as principais diferenças, que a borda acrescenta tamanho ao elemento e que os contornos podem sobrepor outros elementos.

  • outline-style
    • dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden.
  • outline-width
    • thin, medium, thick ou unidades de medida
  • outline-color
    • nome, HEX, RGB, HSL e invert
  • outline-offset: Espaçamento entre a borda e o contorno.
  • outline
    • outline-width
    • outline-style (obrigatório)
    • outline-color
outline: outline-width outline-style outline-color

Código Resultado
div {
border: 3px solid orange;
outline: 3px solid blue;
}
Sou uma div com borda laranja e contorno azul.
div {
border: 3px solid orange;
outline: 3px solid blue;
outline-offset: 20px;
}
Sou uma div com borda laranja e contorno azul. OPS! Observe o azul sobrepondo as linhas da tabela :)
