Layout - Overflow

A propriedade overflow especifica se o conteúdo que ficar extra no documento deve ser cortado ou então adicionada um barra de rolagem para o elemento. O estouro/transbordamento (overflow) funciona apenas se o elemento tiver alguma altura especificada, caso contrário, o elemento por padrão, se ajusta ao tamanho do conteúdo, não havendo estouro/transbordamento.



overflow:visible - transbordamento visível

O valor padrão para o atributo overflow é o visible. Ou seja, todo o conteúdo que ficar fora do elemento, ainda é renderizado ficando visível.

Código Resultado
div {
width: 55%;
height: 77px;
background-color: black;
overflow: visible;
}

overflow:hidden - esconder transbordamento

A propriedade overflow:hidden é usada para cortar todo o conteúdo transbordado.

Código Resultado
div {
width: 55%;
height: 77px;
background-color: black;
overflow: hidden;
}

overflow:scroll - transbordamento com rolagem

A propriedade overflow:scroll é usada para esconder o conteúdo transbordado e acrescentar uma barra de rolagem no elemento, de tal forma que, seja possível ver todo o conteúdo dentro do espaço especificado.

Código Resultado
div {
width: 55%;
height: 77px;
background-color: black;
overflow: scroll;
}

overflow:auto - transbordamento automático

A propriedade overflow:auto é semelhante ao overflow:scroll, entretanto, ele adiciona a barra de rolagem apenas quando for necessário.

background-color: black;

overflow-x overflow-y - transbordamento vertical e horizontal

Também é possível especificar se deseja aplicar as opções anteriores na horizontal(overflow-x) ou na vertical (overflow-y).

Código Resultado

width: 55%;
height: 77px;
background-color: black;
overflow-x: hidden;
overflow-y: scroll;
}
