padding: espaçamento interno

A propriedade padding é usada para criar um espaçamento interno no elemento HTML. Assim como o margin, o padding também, pode gerar um espaçamento interno em cada lado do elemento.

  • padding
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

    border: 1px solid turquoise;
    text-align: center;
}
