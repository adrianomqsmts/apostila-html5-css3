Menus - Barras de navegação

Uma barra de navegação é puramente links em uma lista com diferentes propriedades CSS aplicadas. Então vamos criar, passo a passo uma barra de navegações (Navigation Bar)

1º Passo: Criar uma lista com links.

Código Resultado
<ul>
<li>
<a href="#">Home</a>
<a href="#">Conteúdo</a>
<a href="#">Sobre</a>
<a href="#">Contatos</a>
</li>
</ul>
  • Home
  • Conteúdo
  • Sobre
  • Contatos

2º passo: Remover todos os detalhes visuais da lista, como os marcadores e os espaçamentos.

Código Resultado
ul {
list-style-type: none;
margin: 0;
padding: 0;
}
  • Home
  • Conteúdo
  • Sobre
  • Contatos

3º Passo: Remover a decoração de textos dos links e aplicar uma cor

Código Resultado
ul li a {
text-decoration: none;
color: white;
}
  • Home
  • Conteúdo
  • Sobre
  • Contatos

4º Passo: Fazer com que os item da lista fiquem lado a lados através do display:inline (ou flutuar um ao lado do outro com o float:left).

Código Resultado
ul li {
display: inline;
}

  • Conteúdo
  • Sobre
  • Contatos

5º Passo: Colocar um fundo colorido para a lista

Código Resultado
ul {
list-style-type: none;
margin: 0;
padding: 0;
background-color: green;
}

6º Transformar o link em um elemento de bloco de linha para poder alterar a sua altura e largura

Código Resultado
ul li a {
text-decoration: none;
display: inline-block;
}

7º Passo: Colocar um espaçamento internos nos links para aumentar a área clicável do menu.

Código Resultado
ul li a {
text-decoration: none;
display: inline-block;
padding: 2em;
}
