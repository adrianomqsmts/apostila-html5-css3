Seletores Pseudo-classes

Uma pseudo classe é usada para definir um estado especial de um elemento. Pode se usada para definir um estilo quando o usuário passa o mouse ou mudar o estilo para links visitados e não visitados.

seletor:pseudo-classe {propriedade1:valor1; propriedade2:valor2;..;}


Seletor Descrição
seletor:active Seleciona um elemento especificado ativo. (ex: links)
seletor:checked Seleciona todos os elementos especificados com o atributo checked. (ex: formulários)
seletor:disabled Seleciona todos os elementos especificados com o atributo disabled. (ex: formulários)
seletor:empty Seleciona todos os elementos especificados que não possuem filhos.
seletor:enabled Seleciona todos os elementos especificados que possuem o atributo enabled. (ex: formulários)
seletor:first-child Seleciona o primeiro filho do elemento especificado.
seletor:first-of-type Seleciona a primeira ocorrência do elemento especificado.
seletor:focus Seleciona todos os elementos que estão em foco. (ex. formulários)
seletor:hover Seleciona todos os elementos especificados que estão com o mouse em cima.
seletor:in-range Seleciona todos os elementos especificados que estão dentro dos valores permitidos. (ex: formulários)
seletor:invalid Seleciona todos os elementos que estão com valores inválidos (ex: formulários)
seletor:lang(language) Seleciona todos os elementos especificados que possuem o atributo lang com a linguagem especificada.
seletor:last-child Seleciona todos últimos filhos dos elementos especificados.
seletor:last-of-type Seleciona a última ocorrência do elementos especificado.
seletor:link Seleciona todos os elementos do tipo link que não foram visitados.
:not(selector2) Seleciona todos elementos que não estão especificados.
seletor:nth-child(n) Seleciona o n-ésimo filho do elemento especificado
seletor:nth-last-child(n) Seleciona o último n-ésimo filho do elemento especificado
seletor:nth-last-of-type(n) Seleciona o último n-ésimo tipo do elemento especificado.
seletor:nth-of-type(n) Seleciona o n-ésimo tipo do elemento especificado.
seletor:only-of-type Seleciona todos os elementos do tipo especificados que são filhos únicos
seletor:only-child Seleciona todos os elementos especificados que são filhos únicos
seletor:optional Seleciona todos os elementos especificados com o atributo opcional. (ex formulários)
seletor:out-of-range Seleciona todos os elementos especificados que estão fora dos valores válidos. (ex: formulários)
seletor:read-only Seleciona todos os elementos especificados que são de apenas leitura. (ex: formulários)
seletor:read-write Seleciona todos os elementos especificados que estão que não apenas de leitura. (ex: formulários)
seletor:required Seleciona todos os elementos especificados que são obrigatórios. (ex: formulários)
:target Seleciona o elemento ativo atual. Normalmente especificado na URL através de um ID. (ex: menus e sumários).
seletor:valid Seleciona todos os elementos especificados que possuem valores válidos. (ex formulários)
seletor:visited Seleciona todos os elementos especificados que foram visitados. (ex: link)
