Seletores de pseudo-elementos

Os seletores de pseudo-elementos são usados para selecionarem parte de um elemento.

Seletor Descrição
seletor::after Usado para inserir algum conteúdo após o conteúdo do elemento
seletor::before Usado para inserir algum conteúdo antes do conteúdo.
seletor::first-letter Seleciona a primeira letra de um elemento HTML
seletor::first-line Seleciona a primeira linha de um elemento HTML
seletor::selection Usado para selecionar tudo o que o usuário seleciona.


Código Resultado
p::first-letter {
color red;
}

Oie! eu sou um parágrafo

p::first-line {
color red;
}

Oie! eu sou um parágrafo
eu sou a segunda linha ?

p::selection {
color red;
background-color black;
}

Oie! eu sou um parágrafo, selecionem o meu texto!!
