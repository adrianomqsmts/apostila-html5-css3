Seletores Simples - Nome, identificadores e classes

Os seletores são usados para selecionar elementos HTML que você deseja aplicar algum estilo. Os seleteores são agrupados em 5 categorias:

  • Seletores Simples: Elementos com base no nome, no id e class
  • Seletores de combinação: Elementos baseados em uma relação específica entre eles.
  • Seletores de pseudoclase: Elementos baseados em um determinado estado.
  • Seletores de peseudo-elementos: Seleciona uma parte do elemento para aplicar algum estilo.
  • Seletores de atributos: Seleciona os elementos com base no atributo ou valor do atributo.

Seletores de Elementos CSS

O seletor seleciona elementos HTML com base no nome do elemento. No exemplo abaixo, podemos ver um seletor para o elemento <p>. Assim, todos os elementos <p> da página serão alinhados ao centro e terão um texto na cor vermelha.

<head>
<style>
p {
text-align: center;
color: red;
}
</style>
</head>
<body>
<p> OI! Eu sou um parágrafo. </p>
<p> OI! Eu sou outro parágrafo. </p>
</body>

OI! Eu sou um parágrafo.

OI! Eu sou outro parágrafo.

Seletores de Identificadores CSS

O seletor de id usa o atributo id de um elemento HTML para selecionar um elemento em específico. Um identificador de um elemento tem que ser único na página!

Para selecionar um elemento pelo seu identificador, usamos o caractere hash # (jogo da velha, cerquilha, hastag), seguido pelo valor/nome do indentificador:

#ID {propriedade1:valor; propriedade2:valor2;...;}

<head>
<style>
#paragrafo {
text-align: center;
color: red;
}
</style>
</head>
<body>
<p id="paragrafo"> OI! Eu sou um parágrafo com id paragrafo. </p>
<p> OI! Eu sou outro parágrafo. </p>
</body>

OI! Eu sou um parágrafo com id paragrafo.

OI! Eu sou outro parágrafo.

Seletores de Classes CSS

O seletor de classes seleciona um conjunto de elementos HTML com um atributo de class específico.

Para selecionar um elemento pela sua classe, usamos o caractere ponto . , seguido pelo valor/nome da classe:

.class {propriedade1:valor; propriedade2:valor2;...;}

<head>
<style>
.paragrafos {
text-align: right;
color: green;
}
</style>
</head>
<body>
<p class="paragrafos"> OI! Eu sou um parágrafo com class paragrafos. </p>
<p> Oi! Eu sou somente um parágrafo.</p>
<p class="paragrafos"> OI! Eu sou outro parágrafo com class paragrafos.</p>
</body>

OI! Eu sou um parágrafo com class paragrafos.

Oi! Eu sou somente um parágrafo.

OI! Eu sou outro parágrafo com class paragrafos.

Também é possível aplicar um estilo a uma parte específica de um grupo de classes e não em todos os elementos dessa classe. Digamos que tenhamos dois elementos diferentes usando a mesma classe, mas queremos aplicar o estilo em somente um desses elementos em específico, sem necessáriamente criar uma nova classe.

No exemplo abaixo, queremos que a div tenha letras maiores que as dos parágrafos, então vamos especificar o elemento da classe que queremos aplicar esse estilo.

<head>
<style>
.paragrafos {
text-align: right;
color : green;
}

div.paragrafos {
font-size: 20px;
}
</style>
</head>
<body>
<p class="paragrafos"> OI! Eu sou um parágrafo com class paragrafos. </p>
<div class="paragrafos"> Oi! Eu sou uma div com a class paragrafos. </div>
<div> Oi! Eu sou apenas uma div. </div>
<p class="paragrafos"> OI! Eu sou outro parágrafo com class paragrafos. </p>
</body>

OI! Eu sou um parágrafo com class paragrafos.

Oi! Eu sou uma div com a class paragrafos.
Oi! Eu sou apenas uma div.

OI! Eu sou outro parágrafo com class paragrafos.

Seletore Universal CSS

O seletor universal seleciona todos os elementos HTML de uma página e aplica o estilo a todos eles.

* {propriedade1:valor; propriedade2:valor2;...;}

<head>
<style>
* {
text-align: left;
color : yellow;
}
</style>
</head>
<body>
<p> Oi! Eu sou um parágrafo.</p>
<div> Oi! Eu sou uma div.</div>
<h1> Oi! Eu sou um cabeçalho H1.</h1>
</body>

Oi! Eu sou um parágrafo.

Oi! Eu sou uma div.

Oi! Eu sou um cabeçalho H1.

Seletore de Agrupamento CSS

O seletor de agrupamento seleciona mais de um elemento HTML ao mesmo tempo e aplica o estilo em todos eles. Essa seleção minimiza o código de ter que fazer a mesma coisa para diferentes elementos.

elemento1, elemento2, ... {propriedade1:valor; ...;}

<head>
<style>
p, div, h1 {
text-align: left;
color: yellow;
}
</style>
</head>
<body>
<p> Oi! Eu sou um parágrafo. </p>
<div> Oi! Eu sou uma div. </div>
<h1> Oi! Eu sou um cabeçalho H1. </h1>
</body>

Oi! Eu sou um parágrafo.

Oi! Eu sou uma div.

Oi! Eu sou um cabeçalho H1.
