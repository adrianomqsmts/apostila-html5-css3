Podemos usar o o seletor :hover para destacar a linha ao passar o mouse. 
    table {
        border-collapse collapse;
        height: 100%;
        wight: 100%;
    }    
    table td, table th {
        padding: 1.3em;
    }
    table tr:hover {
        background-color: black;
    }

Para fazer uma tabala listada, usamos o seletor :nth-child(even) especificando a cor do background. (even - linhas impar) 
    table {
        border-collapse collapse;
        height: 100%;
        wight: 100%;
    }    
    table td, .ex7 th {
        padding: 1.3em;
    }
    table:nth-child(even) {
        background-color: gray;
    }
    table, table tr th, table tr td, table tr {
        border: green;
    }
    table tr:hover {
        background-color: black;
