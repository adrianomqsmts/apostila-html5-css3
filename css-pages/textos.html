color: cores dos textos

A propriedade color é usada para definir a cor do texto, podendo assumir valores como o nome de uma cor, HEX ou RGB. Entretanto, de acordo com as normas, sempre que definir uma cor, deve-se usar a propriedade background-color. 

 p  {
    color: orange;
    background-color: none;
}

Oi! Voltei. Eu sou um parágrafo com letras laranjas e fundo vazio. 

 p  {
    color: green;
    background-color: black;
}

Oi! Voltei. Eu sou um parágrafo com letras verdes e fundo preto.

text-align: alinhamento de textos

A propriedade text-align e vertical-align são usadas para definir a o alinhamento horizontal e o alinhamento vertial, respectivamente, de um texto.

O text-align pode assumir valores como left (esquerda), right (direita), center (centralizado) e justify (justificado). 

p  {
    text-align: right;
}

text-decoration: underline, sobrelinha e tachado.

A propriedade text-decoration é usada para escrever linhas em textos ou remover, através do valor none, essas linhas, como nos casos dos links. 

p  {
    text-decoration: overline;
}

Olhe, estou com overline (sobrelinha). 

p  {
    text-decoration: line-through;
}

Olhe, estou com line-through (tachado). 

p  {
    text-decoration: underline;
}

Olhe, estou com underline (sublinhado).

text-transform: minúsculo, maiúsculo e capitalizado

A propriedade text-transform é usada para transformar todas as letras do texto em maiúsculas (uppercase), minúsculas (lowercase), ou as primeiras letras de cada palavra em maiúscula (capitalize).

Olhe! Eu sou apenas um parágrafo. 

p  {
    text-transform: lowercase;
}

Olhe! Eu sou apenas um parágrafo. 

p  {
    text-transform: uppercase;
}

Olhe! Eu sou apenas um parágrafo. 

p  {
    text-transform: capitalize;
}

Olhe! Eu sou apenas um parágrafo.
