Transições

As transições são usada para que se altere o estilo de um elemento de forma suave durante um determinado tempo especificado.

A propriedade transition precisa de dois valores, a primeira é qual outra propriedade você deseja aplicar no elemento, seguido pelo tempo que essa propriedade especificada deve transitar.

O próximo passo é utilizar o seletor :hover com a propriedade especificada na propriedade transition

div {
background-color: darkgreen;
width: 50px;
height: 200px;
transition: width 3s;
}
div:hover {
width: 100%;
}
Passe o Mouse

Podemos especificar a qual tipo de aceleração queremos para transição com o atributo transition-timing-function que possuí os seguintes valores :

  • ease: Especifica um efeito de transição com início lento, depois rápido e, a seguir, fim lento (é o padrão).
  • linear: Especifica um efeito de transição com a mesma velocidade do início ao fim.
  • ease-in: Especifica um efeito de transição com um início lento.
  • ease-out: Especifica um efeito de transição com um final lento.
  • ease-in-out: Especifica um efeito de transição com início e final lento.
div {
background-color: darkgreen;
width: 50px;
height: 200px;
transition: width 3s;
transition-timing-function: linear;
}
div:hover {
width: 100%;
}
Passe o Mouse
div {
background-color: darkgreen;
width: 50px;
height: 200px;
transition: width 3s;
transition-timing-function: ease-in-out;
}
div:hover {
width: 100%;
}
Passe o Mouse

Podemos especificar o tempo de demora em que vai iniciar a transição com a propriedade transition-delay.

div {
background-color: darkgreen;
width: 50px;
height: 200px;
transition: width 3s;
transition-timing-function: ease-in;
transition-delay: 1s;
}
div:hover {
width: 100%;
}
Passe o mouse, demora 1 s.

Podemos usar transition-property e transition-duration no lugar de transition, assim como também podemos usar as demias propriedadas de transições em transition.

div {
background-color: darkgreen;
width: 50px;
height: 200px;
transition-property: width;
transition-duration: 3s;
transition-timing-function: ease-out;
transition-delay: 1s;
}
div:hover {
width: 100%;
}
Passe o mouse, demora 1 s.
div {
background-color: darkgreen;
width: 50px;
height: 200px;
transition: width 3s ease-out 1s;
}
div:hover {
width: 100%;
}
Passe o mouse, demora 1 s.

Se quisermos usar a transição em mais de uma propriedade, podemos usar outra declaração transition ou separar por vírgulas cada propriedade.

transition: propriedade duração efeito demora, propriedade duração efeito demora...;

div {
background-color: darkgreen;
width: 50px;
height: 50px;
transition: width 3s ease-out 1s, height 3s linear 0s;
}
div:hover {
width: 100%;
height: 200px;
background-color: green;
color: black; }
Passe o mouse, demora 1 s.
