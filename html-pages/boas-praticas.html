Padrão de codificação

Sempre que for codificar uma página em HTML sigam esses padrões.

  • Sempre declare o tipo do documento. No caso do HTML5, declare:
    • <!DOCTYPE html>
  • Usem nome de elementos sempre em minúsculas como <a> e <p> evitando <A> ou <P>.
    • <a>
  • Feche todos os elementos.
    • <p></p>
    • <br/>
  • Use sempre os nomes dos atributos em minúsculas como <a href="url"></a> ao invés de <a HREF="url"></a>.
    • <a href="url"></a>
  • Sempre declares os valores dos atributos e entre aspas duplas.
    • <a href="url"></a>
  • Sempre especifique, através de CSS, a altura, largura e o atributo alt das imagens, para evitar má formatação das imagens.
    • <img href="url" style="width:128px;height:128px" alt="img">
  • Evite colocar espaçamentos entre os atributos e seus valores (atributo="valor").
    • <a href="url"></a>
  • Evite linhas longas de código, sempre identando corretamente o códifo fonte. Não é comveniente que tenha barra de rolagem lateral no documento.
    • Existem IDEs que permitem o recurso de wrap, busque usá-lo.
  • Evite linhas em branco e recuos sem motivos. Use-os apenas para separar blocos de códigos grandes ou blocos lógicos. Evitadando o uso da tecla TAB, preferindo dois espaços para identar os elementos.
  • Nunca ignore o elemento <title>, pois é muito importante para buscadores de pesquisas.
  • Fechar elementos vazios é opcional, mas necessário caso algum software xml ou xhtml use a sua página.
  • Inclua o atributo lang dentro da tag <html lang="pt-br">.
  • Sempre declare os metadados o mais cedo possível, impedindo erros de interpretação.
    • <meta charset="utf-8">.
  • Configure sempre a janela de vizualização através do elemento <meta>.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">.
      • A width=device-width define a largura da página para seguir a largura da tela do dispositivo (que irá variar dependendo do dispositivo).
      • A initial-scale=1.0 define o nível de zoom inicial quando a página é carregada pela primeira vez pelo navegador.
  • SEMPRE COMENTE O CÓDIGO FONTE HTML
    • <!-- COMENTE!!! -->
  • Use sempre nomes de arquivos em letras minúsculas, sempre declarando suas devidas extensões.
    • arquivo.extensao
  • A pagina web principal deve se chamar index.
    • index.html
    • index.htm
