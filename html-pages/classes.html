Classes (.)

O atributo class é usado para especificar uma classe para um elemento HTML, sendo útil para quem deseja aplicar estilos CSS (ou funções javascript) a um determinado grupo de elementos diferentes ( que podem ser p, h1-h6, div, span, etc ao mesmo tempo).

<div class="azul">
BLOCO AZUL
</div>
<div class="verde">
BLOCO VERDE
</div>
<div class="azul">
BLOCO AZUL
</div>

<style>
.azul {
border: 5px solid blue;
}
.verde {
border: 5px solid green;
}
</style>
BLOCO AZUL
BLOCO VERDE
BLOCO AZUL

Um elemento pode ter mais de um classe separada por espaços e o nome de uma classe não pode começar com um número.

<div class="primeira segunda terceira"></div>
