CSS BÁSICO

O CSS é usado para formatar o layout e estilo de uma página web, podendo controlar, cores, fontes, tamanhos, dentre outras diversas possibilidades de estilos. Além disso, ele ajuda a evitar repetição de códigos e deixa a estrutura HTML mais limpa se usado de forma externa.

Para usar o CSS somente em um elemento, precismos usar o atributo style=" " dentro da tag inicio do elemento que queremos aplicar o estilo. Cada característica deve ser separada por ponto-vírgula e finalizado com o mesmo.

<p style="color: red; background-color: black;">
parágrafo
</p>

parágrafo

Para usar o CSS em um grupo de elementos do mesmo tipo, precismos usar a tag <style>, dentro da tag <head>. Respeitando a sintaxe: elemento { propriedade: valores; }

<head>
<style>
p {
color: white;
background-color: blue;
}
</style>
</head>
<body>
<p> parágrafo1 </p>
<p> parágrafo2 </p>
</body>
