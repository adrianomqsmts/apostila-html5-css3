Elementos Block and Inline

Um elemento de nível de bloco sempre começa em uma nova linha e ocupa toda a largura disponível (se estende para a esquerda e para a direita, tanto quanto pode). O mais utilizado para definir blocos, é o elemento <div> que funciona como um contêiner de elementos. Entretanto, outras tags também são de blocos em nível, como: <address> <article <aside> <blockquote> <canvas> <dd> <div> <dl> <dt> <fieldset> <figcaption> <figure> <footer> <form> <h1>-<h6> <header> <hr> <li> <main> <nav> <noscript> <ol> <p> <pre> <section> <table> <tfoot> <ul> <video>.

Um elemento embutido (inline) não começa em uma nova linha e só ocupa a largura necessária e não podem conter um bloco de nível. O mais útilizado para blocos embutidos, é o elemento <span> que funciona como um contêiner de elementos. Entretanto, outras tags também são de blocos embutidos, como: <a> <abbr> <b> <bdo> <br> <button> <cite> <code> <dfn> <em> <i> <img> <input> <kbd> <label> <map> <object> <output> <q> <samp> < script> <select> <small> <span> <strong> <sub> <sup> <textarea> <tt> <var>
