HEAD

Como vimos, o <head> é um elemento de meta dados, servindo de contêiner para outros elementos, como o <title>, <style>, <meta>, <link>, <script> e <base>. Esses elementos são uma descrição sobre o documento HTML e não é visível para os usuários.

title

O elemento <title> define o título do documento, devendo ser somente texto, e é mostrado na barra de título do navegador. Importante ressaltar que o título deve ser significativo para a página, pois o mesmo é usado por mecanismos de buscas para encontrar a sua página e exibíla nos resultados de busca.

style

O elemento <style> como vimos, é usado para definir informações sobre o estilo da página, através de sintaxe CSS ou JavaScript.

link

O elemento <link> é usado para definir uma relaçao entre o documento atual e um documento externo, como, por exemplo, um arquivo CSS.

<link rel="stylesheet" href="estilo.css">

meta

O elemento <meta> é normalmente usado para especificar o conjunto de caracteres, palavras-chaves, autor do documento, descrição da página, além de poder configurar a janela de visualização. Essas informações são usadas pelo navegador, mecanismos de pesquisas e outros serviços web.

<meta charset="UTF-8">

<meta name="description" content="HTML Básico e Grátis Totalmente Pago">

<meta name="keywords" content="HTML, CSS, JavaScript">

<meta name="author" content="Meu Nome">

(atualiza a página de 30 em 30 segundo)
<meta http-equiv="refresh" content="30">

charset

É o atributo do elemento <meta> e é usado para definir qual devem ser os conjuntos de caracteres, exibidos corretamente por um navegador. Sendo o mais comum, o UTF-8. Mais sobre charset

script

O elemento <script> é usado para definir o JavaScript do lado do cliente, assim como o style define o estilo com o CSS.

base

O elemento <base> é usado para especificar o URL e/ou destino de todos os URLs relativos em uma página, evitando assim, digitar todo o endereço da pagina atual para nevegar entre os documentos.

<base href="https://paginaweb.com" target="_blank">
