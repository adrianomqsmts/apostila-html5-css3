HTML em HTML

Iframes

Um iframe é um elemento do HTML utilizado para incorporar outra página da web na página atual. Ou seja, com ele você pode incluir conteúdos de outras páginas na sua própria página como vídeos, imagens, corpo de outras páginas e publicidades.

<iframe src="url" title="descrição do conteúdo">

Por exemplos, podemos abrir o youtube, clicar sobre um vídeo com o botão direito, procurar a opção "incorporar códido de incorporação", que a própria plataforma irá disponibilizar o código HTML, e ent'ao basta introduzir o vídeo em sua página.

Código Resultado
<iframe width="50%" height="410" title="Video da música wonder do Shawn Mendes" src="https://www.youtube.com/embed/fHeQemJJQII" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen> </iframe>
<iframe src="iframe-fixed.html" style="width:100%; border:4px solid #e5e5e5"> </iframe>


Object

O elemento <object> foi projetado para incorporar plug-ins (como applets Java, leitores de PDF e Flash Players) em páginas da web, mas também pode ser usado para incluir HTML em HTML:

Código Resultado
<object style="width: 100%; height: 200px;" data="introducao.html"></object>


embed

O elemento <embed> também define um objeto incorporado em um documento HTML. Mas não contém a tag de fechamento.

Código Resultado
<embed style="width: 100%; height: 200px;" src="listas.html">
