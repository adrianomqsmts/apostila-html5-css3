Identificadores (#)

O identidicador id é um atributo usado para especificar um elemento, sendo útil para aplicar estilos únicos a um único elemento, sem que os demais sejam alterados. Além disso é possível usar links para navegar entre os identificadores como um menu ou sumário.

<div id="verde"> BLOCO ID VERDE </div>
<div> BLOCO </div>
<div> BLOCO </div>

<style>
div {
border: 5px solid blue;
}
#verde {
border: 5px solid green;
}
</style>
