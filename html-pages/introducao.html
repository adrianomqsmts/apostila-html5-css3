INTRODUÇÃO

HTML (Hyper Text Markup Language) é uma linguagem de marcação de hipertexto,padrão para a criação de páginas daWeb, que descreve a estrutura das páginas da Web usando marcações (elementos/tags).

O HTML surge apenas para estruturar de forma semântica a página, se limitando apenas a estruturar páginas, onde toda a forma de aplicação de estilos usando HTML deve ser evitada, deixando para o CSS cuidar de todo o estilo da página.



Elementos

O elemento HTML é tudo, desde a tag inicial (marcador de inicio) até a tag final (marcador de finalização), podendo ter elementos aninhados, ou seja, elementos que contém outros elementos dentro de suas marcações.

Um elemento tem a seguinte sintaxe:

<tagname> CONTEÚDO </tagname>

Importante!!! Nunca se esqueça de colocar uma tag final aos elementos, mesmo que eles sejam exibidos corretamente. Pois, resultados inesperados e erros podem ocorrer se você esquecer de finalizar uma marcação.

Entretanto, existem tags chamadas de elementos vazios (ou selfclosing). Esses elementos vazios não possuem conteúdo e normalmente não precisam ser fechadas, pois a maioria dos navegadores a reconhecem dessa forma (em XTHML é fechada assim <tagname/>).Um exemplo simples, é uma quebra de linha <br>.

Como último detalhe, o HTML não diferencia letras maiúsculas e minúsculas (no case sensitive), ou seja, uma tag pode ser escrita tanto como <TAG> quanto <tag>, que serão aceitas. Entretando é importante manter um padrão de uso, e é fortemente recomendado que usem tag em minúsculas.



Atributos

Os atributos HTML tem a função de fornecer informações adicionais sobre os elementos HTML. Eles são especificados dentro da tag inicial comO um par nome/valor, com a sintaxe:

nome = "valor"

Atributos são informações que passamos para um elemento para que ele se comporte de uma maneira que seja esperada. Existem dois tipos de atributos, os atributos globais que funcionam em todas os elementos e os atributos específicos que são direcionados para cada elemento.

Como exemplo, vamos usar um atributo para na tag <p>. O atributo title define uma informação extra sobre o elemento, exibindo ao passar o mouse sobre o elemento, o texto especificado.

Código Resultado
<p title="isso é um parágrafo">
Conteúdo do parágrafo. Passe o mouse aqui.
</p>

Conteúdo do parágrafo. Passe o mouse aqui.



Estrutura básica de um site HTML

Para começar a escrever um arquivo do tipo HTML5, antes, ele deve seguir a seguinte estrutura base, que será o pilar de toda a esturua da página:

<!DOCTYPE html>
<html>
<head>
<title>Título da página</title>
</head>

<body>
</body>
</html>
  • O elemento <!DOCTYPE html> define que este documento é um documento HTML5
  • O elemento <html> é a raiz de uma página HTML
  • O elemento <head> é quem contém as metainformações sobre a página HTML
  • O elemento <title> especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página)
  • O elemento <body> define o corpo do documento, sendo um contêiner (estrutura) para todo o conteúdo visível, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.
