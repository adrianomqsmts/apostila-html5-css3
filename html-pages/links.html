LINKS

Os links são usados para que os usários possam atráves de uma interação (normalmente um clique, ir de uma página para outra. Geralmente um link é destacado, de tal forma que quando um mouse é posto sobre ele, o ícone é transformado em uma pequena mão. Um link pode ter seu conteúdo ligado a um texto, uma imagem ou qualquer outro elemento HTML.

Um link é defindo pela tag <a>, onde o atributo href especifica a URL ou caminho do arquivo/página.

<a href="https://google.com">
   Texto do link
</a> 		Texto do link
<a href="https://wwww.google.com">
   <img src="imagens/img.png" style="width:100%; height:200" alt="PUMP">
</a>

O atributo title de um link adciona informações extras sobre um elemento, onde as informações geralmente são mostradas ao o mouse sobre o link.

<a href="https://google.com" title="Site do Google">
   Texto do link, passe o mouse!
</a> 		Texto do link, passe o mouse!
