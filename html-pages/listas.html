LISTAS

Listas Não Ordenadas

A tag <ul> (unordered list) é a tag usada para delimitar uma lista não ordenada, enquanto a tag <li> (list item) indica os itens dentro da lista. Onde, por padrão, os itens da lista serão marcados com marcadores de pequenos circulos.

Código Resultado
<ul>
<li> Item 1 </li>
<li> Item 2 </li>
<li> Item 3 </li>
</ul>
  • Item 1
  • Item 2
  • Item 3

Os principais atributos relacionados a tag <ul> estão ultapassados, e devem ser evitado, escolhendo um equivalente em CSS (type por list-style-type, por exemplo).



Listas Ordenadas

A tag <ol> (ordened list) é a tag usada para delimitar uma lita ordenada, enquanto a tag <li> (list item) ordena os itens dentro da lista. Por padrão, os números são usados para marcar os itens.

Código Resultado
<ol>
<li> Item 1 </li>
<li> Item 2 </li>
<li> Item 3 </li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Para criar listas aninhadas, precisamos colocar dentro da tag <li> outra tag de lista, criando assim, uma lista de lista, que será reconhecida pelo navegador como um novo nível de lista. Observe que a tag de fechamento </li> não é colocada imediatamente após o texto, e sim ao final do fechamento da nova lista.

Código Resultado
<ol>
<li> Item 1 <li>
<li> Item 2
<ul>
<li>Subitem 2.1 </li>
<li> Subitem 2.2 </li>
<li> Subitem 2.3 </li>
</ul>
</li>
<li> Item 3
<ol>
</li> Subitem 3.1 </li>
</li> Subitem 3.2 </li>
</ol>
</li>
<ol>
  1. Item 1
  2. Item 2
    • Subitem 2.1
    • Subitem 2.2
    • Subitem 2.3
  3. Item 3
    1. Subitem 3.1
    2. Subitem 3.2
