PARÁGRAFOS

Um parágrafo sempre começa em uma nova linha e geralmente, seu conteúdo, é um bloco de texto. Os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.

Para demilitar um parágrafo em HTML usamos a tag <p>(paragraph).

<p>
Aqui
tem um


parágrafo...
<p>

Aqui tem um parágrafo...

Com o HTML, todos os espaçoes e quebras de linhas usando o ENTER são ignorados dentro da tag <p>. E para contornar isso, temos algunas possibilidades.

A primeira dela é usando a tag <br> (break), que gera uma quebra de linha simples, sem espaçamento entre os elementos. A tag <br> é uma tag vazia (Selfclosing), o que significa que não é necessário escrever a tag final.

<p> <br>
Aqui <br>
tem um <br><br><br> parágrafo...
</p>

Aqui
tem


um parágrafo...

A segunda forma é usar a tag <pre> que define um texto preformatado, exibindo todo o texto delimitado pelas tags de forma que os espaçamentos sejam respeitados. Mas cuidado com a identação, porque isso pode gerar mais espaços que o necessário.

<pre>
Aqui
tem um


parágrafo...
</pre> 
Aqui
tem um



parágrafo...

Podemos usar a tag <hr> (horizontal rules) para criar uma linha horizontal que separar conteúdos em uma página HTML. A tag <hr> é uma tag vazia (Selfclosing), o que significa que não é necessário escrever a tag final.

<hr>
