Elementos Semânticos

Elementos semânticos dizem algo sobre o seu conteúdo, e isso diferem dos elementos div e span, que não especificam nada sobre o conteúdo.



section

O elemento <section> define uma seção do documento, que é descrita pela documentação da W3C como um agrupamento temático de conteúdo, normalmente com um título. Como por exeplo, uma seção para introdução, conteúdo e contatos.



article

O elemento <article> especifica um conteúdo independente e completo, como postagens ou artigos de jornais.



header

O elemento <header> funciona como um contêiner para o conteúdo introdutório ou um conjunto de links de navegação, podendo conter, um título, logotipo e informações de autoria.



footer

O elemento <footer> define um rodapé para uma seção ou para um documento, e contém, normalmente, o mapa do site, os links principais, direitos utorais, as informações de contatos e os documentos relacionados.



nav

O elemento <nav> define um conjunto de links de navegação, geralmente são os menus.



aside

O elemento <aside> define um conteúdo relacionado ao conteúdo próximo de forma complementar eF geralmente fica na lateral do conteúdo principal.



main

O elemento <main> especifica o conteúdo principal de maior relevância dentro da página.



Estrutura final de uma página semântica

<!DOCTYPE html>
<html>
<head>
<title>Título da página</title>
</head>
<body>
<header>
<nav></nav>
</header>
<main>
<section>
<article>
<h1> Titulo </h1>
<p> Lorem ipsum dolor sit amet consectetur.<p>
</article>
</section>
</main>
</body>
</html>


Outros elementos semânticos

  • time: O elemento <time> é utilizado para representar datas. Assim, caso seja necessário informar a data em que um conteúdo foi escrito, podemos declarar a tag <timeF> e acrescentar a ela o atributo datetime para escrever a data de forma padronizada.
