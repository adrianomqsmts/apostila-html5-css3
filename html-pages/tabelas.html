TABELAS

As tabelas permitem que os dados sejam organizados em linhas e colunas. O corpo de uma tabela é definido com a tag <table>.

  • <tr>: Demarca as linhas de uma tabela.
  • <td>: Demarca as colunas de uma tabela em uma linha.
  • <th>: Demarca os títulos das colunas em um linha.
  • <caption>: Especifica a legenda de uma tabala.
<table>
<caption> Tabela de Nomes </caption>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Fulano da Silva</td>
<td>70</td>
</tr>
<tr>
<td>Ciclano de Sousa</td>
<td>27</td>
</tr>
<tr>
<td>Beltrano Das Neves</td>
<td>9</td>
</tr>
</table>

Para formatar um tabela, deve-se usar o CSS. Assim, para colocarmos linhas, vamos usar o atributo style="border: 1px solid black" para melhor visualizar a mesclagem de células.

Mesclar Células

Para fazer com que uma célula ocupe mais de uma coluna, é usado o atributo colspan seguido pela quantidade de colunas mescladas. Observe o último elemento </td colspan="2">

<table>
<caption> Tabela de Nomes </caption>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Fulano da Silva </td>
<td>70</td>
</tr>
<tr>
<td colspan="2" style="border: 1px solid black">Ciclano de Sousa</td>
</tr>
</table>
Tabelas de nomes
Nome Idade
Fulano da Silva 70
Ciclano de Sousa

Para fazer com que uma célula ocupe mais de uma linha, é usado o atributo rowspan seguido pela quantidade de colunas mescladas. Observe que usamos <td rowspan="2"> na idade 70

<table>
<caption> Tabela de Nomes </caption>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Fulano da Silva </td>
<td rowspan="2" style="border: 1px solid black">70</td>
</tr>
<tr>
<td>Ciclano de Sousa</td>
</tr>
</table>
Tabelas de nomes
Nome Idade
Fulano da Silva 70
Ciclano de Sousa

Tabelas Semânticas

Os elementos <thead>, <tbody> e <tfoot> podem ser usados para especificar cada parte de uma tabela, cabeçalho, corpo e rodapé respectivamente.

Os navegadores podem usar esses elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho e rodapé, além de facilitar o trabalhos de leitores de telas, e a formatação na hora da montagem da impressão.

<table>
<caption>Tabela</caption>
<thead>
<tr>
<th>Coluna 1</th>
<th>Coluna2</th>
</tr>
</thead>
<tbody>
<tr>
<td>Dado 1</td>
<td>Dado 2</td>
</tr>
</tbody>
<tfoot>
<tr>
<td>Resultado 1</td>
<td>Resultado 2</td>
</tr>
</tfoot>
</table>
Tabela
Coluna 1 Coluna 2
Dado 1 Dado 2
Resultado 1 Resultado 2
