Arrays em JavaScript

Arrays são usados para guardar em um vetor que começa com a posição 0, diferentes tipo de variáveis "novoElemento" ou objetos. Para declarar uma array basta colocar entre [] cada um dos elementos que irão fazer parte do array.

var array[ "Oi", "novoElemento", true];

Para acessar o primeiro elemento do array, basta digitar array[0], enquanto que para acessar o segundo basta digitar array[0], e assim sucessivamente.

Mostrar Código

<div id="array1"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
document.getElementById("array1").innerHTML = array[0];
</script>


split() - Transformando string em array

O método string.split("separador") quebra a string em várias string menores e a transforma em um array de N posições, onde cada posição terá uma string. Um exemplo simples seria a string abc-def-erg com o método string.split("-") que iria se tornar um array do tipo [abc,def,erg], onde a primeira posição array[0] estaria o elemento abc

Mostrar Código

<div id="split"></div>
<script>
var texto = "Meu nome é Fulano";
var array = texto.split(" ");
document.getElementById("split").innerHTML = "O primeiro elemento do array : " + array[0] + "<br> O segundo elemento do array : " + array[1];
</script>

length : tamanho do array

Podemos saber o tamanho do array através da propriedade array.length que irá retornar a quantidade de elementos dentro do array.

Mostrar Código

<div id="length"></div>
<script>
var array = [ "elemento1", "elemento2", "elemento3"];
var tamanho = array.length;
document.getElementById("length").innerHTML = tamanho
</script>


Muitas vezes podemos querer acessar o último elemento de um array, para isso usamos a propriedade length e subtraímos 1, pois a posição inicial do vetor é 0. Por exemplo, um array com 10 elemento, o último elemento está na posição 9. array[array.length - 1]

Mostrar Código

<div id="length2"></div>
<script>
var array = [ "elemento1", "elemento2", "elemento3"];
var ultimoElemento = array[array.length - 1];
document.getElementById("length2").innerHTML = ultimoElemento
</script>


push() : Inserindo um elemento na última posição de um array

O método array.push(novoElemento) insere ao final do array um novo elemento.

Mostrar Código

<div id="push"></div>
<script>
var array = [ "elemento1", "elemento2", "elemento3"];
array.push("novoElemento");
document.getElementById("push").innerHTML = array
</script>


O método array.push("novoElemento") retorna um valor in"novoElemento"o com o novo tamanho da string que pode ser armazenado em uma variável

Mostrar Código

<div id="push2"></div>
<script>
var array = [ "elemento1", "elemento2", "elemento3"];
var novoTamanho = array.push("novoElemento");
document.getElementById("push2").innerHTML = novoTamanho
</script>


pop() : Removendo o último elemento em um array

O método array.pop() remove o último elemento do array e retorna o valor do último elemento, que pode ser armazenado em uma variável.

Mostrar Código

<div id="pop"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
var elementoRemovido = array.pop();
document.getElementById("pop").innerHTML ="O elemento removido foi: " + elementoRemovido + " - Ficando : " + array;
</script>


toString() : Convertendo o array em uma string

O método array.toString() converte todos os elementos dentro de um array em uma string separada pelo caractere de vírgula , . Por padrão digitar array ou array.toString() tem o mesmo resultado

Mostrar Código

<div id="toString"></div>
<script>
var array = [ "elemento1", 'elemento2', "elemento3"];
document.getElementById("toString").innerHTML = array.toString();
</script>


join() : Convertendo o array em uma String

O método array.join("separador") faz a mesma coisa que o método toString(), com a diferença que você pode especificar como quer separar cada elemento dentro do array na nova string.

Mostrar Código

<div id="join"></div>
<script>
var array = [ "elemento1", 'elemento2', "elemento3"];
document.getElementById("join").innerHTML = array.join(" - ");
</script>


shift() : Removendo o primeiro elemento de um array

O método array.shift() remove o primeiro elemento de um array e ainda desloca todos os demais elementos para que o segundo elemento assuma a primeira posição, e assim sucessivamente. O método retorna o valor do primeiro elemento que pode ser armazenado em uma variável.

Mostrar Código

<div id="shift"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
var elementoRemovido = array.shift();
document.getElementById("shift").innerHTML ="O elemento removido foi: " + elementoRemovido + " - Ficando : " + array;
</script>


unshift() : Inserindo um elemento na primeira posição de um array

O método array.unshift("novoElemento") insere o novo elemento na posição inicial do array, e descola o antigo primeiro elemento para a segunda posição e assim sucessivamente. O método retorna o novo tamanho do array que pode ser armazenado em uma variável.

Mostrar Código

<div id="unshift"></div>
<script>
var array = [ "elemento1", "elemento2", "elemento3"];
var novoTamanho = array.unshift("novoElemento");
document.getElementById("unshift").innerHTML = "Novo tamanho : " + novoTamanho + " ficando: " + array.join(" - ");
</script>


Alterando qualquer elemento de um array

Para poder alterar um elemento de uma matriz basta usar a notação array[posição] = novoElemento que iremos alterar o elemento da posição especificada pelo valor atribuído.

Mostrar Código

<div id="alterando"></div>
<script>
var array = [ "elemento1", "elemento2", "elemento3"];
array[1] = "novoElemento"
document.getElementById("alterando").innerHTML = array.join(" - ");
</script>


splice() : Removendo e adicionando elementos em qualquer posição de um array

O método array.splice(posiçãoInserir, quantidadeRemover, novoElemento1, novoElemento2...) é usado para fazer várias operações ao mesmo tempo, como inserir, remover e até alterar elementos de um array. Para isso o método precisa de pelo menos dois parâmetros, o primeiro especifica em qual posição deseja inserir os elementos, o segundo parâmetro especifica quanto elementos você quer remover a partir daquela posição, seguido por nenhum ou vários elementos que deseja inserir. A função retorna os valores de todos os itens removidos em um array.

Removendo elementos de qualquer posição

Mostrar Código

<div id="spliceRemove"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
var elementosRemovidos = array.splice(1,2);
document.getElementById("spliceRemove").innerHTML ="O elemento removido foi: " + elementosRemovidos + " - Ficando : " + array;
</script>


Alterando um elemento de qualquer posição

Mostrar Código

<div id="spliceAlterar"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
var elementosRemovidos = array.splice(1,1, "NovoElemento");
document.getElementById("spliceAlterar").innerHTML ="O elemento removido foi: " + elementosRemovidos + " - Ficando : " + array.join(" - ");
</script>


Inserindo elementos em qualquer posição

Mostrar Código

<div id="spliceInserir"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
var elementosRemovidos = array.splice(1,0, "NovoElemento1", "NovoElemento2", "NovoElemento3");
document.getElementById("spliceInserir").innerHTML = array.join(" - ");
</script>


Inserindo, alterando e removendo elementos em qualquer posição

Mostrar Código

<div id="splice"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
var elementosRemovidos = array.splice(1, 2, "NovoElemento1", "NovoElemento2", "NovoElemento3");
document.getElementById("splice").innerHTML ="O elemento removido foi: " + elementosRemovidos + " - Ficando : " + array.join(" - ");
</script>


concat() : Unindo dois ou mais arrays

O método array.concat(array1, array2, ...) é usado para concatenar dois ou mais arrays (Também pode receber uma string), inserindo o array do parâmetros ao final do array. Lembrando que o método gera um novo array com os valores concatenados e não altera o array original.

Mostrar Código

<div id="concat"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3"];
var array2 = ["elemento4", "elemento5"];
var array3 = ["elemento6"];
var novoArray = array.concat(array2, array3);
document.getElementById("concat").innerHTML = novoArray.join(" - ");
</script>

slice(): Cortando um array

O método array.slice(posiçãoInicial, posiçãoFinal) corta o array na posição especificada retornando um novo array com os elementos dentro do intervalo (Posição Final não inclusa). Caso a posição Final estava ocultada, então a string é cortada até o final.

Mostrar Código

<div id="slice"></div>
<script>
var array = ["elemento1", "elemento2", "elemento3", "elemento4"];
var novoArray = array.slice(1, 3);
document.getElementById("slice").innerHTML = novoArray.join(" - ");
</script>

sort(): Ordenando alfabeticamente os valores de um array

O método array.sort() é usado para ordenar alfabeticamente os elementos dentro de um array (não funciona com números).

Mostrar Código

<div id="sort"></div>
<script>
var array = ["Cereja", "Abacate", "Morango", "Manga", "Goiaba", "Amora"];
var novoArray = array.sort();
document.getElementById("sort").innerHTML = novoArray.join(" - ");
</script>

reverse() : Revertendo a ordem dos elementos

O método array.reverse()inverte a ordem em que os elementos estão dentro do array, ou seja, o primeiro elemento vai para a última posição, o segundo elemento vai para a penúltima posição, e assim sucessivamente.

Mostrar Código

<div id="reverse"></div>
<script>
var array = ["Cereja", "Abacate", "Morango", "Manga", "Goiaba", "Amora"];
var novoArray = array.reverse();
document.getElementById("reverse").innerHTML = novoArray.join(" - ");
</script>

Ordenando valores numéricos de um array

Para ordenar os elementos numéricos de um array, precisamos do auxílio de uma função como parâmetro dentro do método sort(function). Essa função é praticamente padrão, onde basicamente retorna a diferença entres os elementos, dizendo qual elemento é maior ou menor. Para saber mais acesse :Referência do método sort()

function(a, b){return a-b}

Mostrar Código

<div id="sortNum"></div>
<script>
var array = [100, 50, 60, 10, 80, 0, 1, 101 ];
var novoArray = array.sort(function(a, b){return a-b});
document.getElementById("sortNum").innerHTML = novoArray.join(" - ");
</script>

Maior e menor valor em um array

Depois de ordenar um array o menor valor estará na primeira posição de um array array[0], enquanto o maior valor estará na última posição de um elemento array[array.length - 1]. Outra forma de encontrar o maior ou menor valor de um array é usando a biblioteca MATH através dos métodos Math.max e Math.min
