RegExp (expressões regulares) em JavaScript

Expressões regulares são é um conceito um pouco avançado, mas que remete a uma sequência de caracteres que formam um padrão de pesquisa. Basicamente podemos formar expressões para encontrar em pesquisas palavras que começem de terminem com uma terminada letra, que tenham pelo menos 5 letras idênticas e diversas outras possibilidades.

Modificadores

Primeiro iremos usar uma substituição padrão com o método texto.replace() da string.

Mostrar Código

<div id="replace0"></div>
<script>
var txt = "Ameixa Banana amora goiaba banana melancia melão BANANA maçã banana";
var novoTxt = txt.replace("banana", "QUEIJO");
document.getElementById("replace0").innerHTML = "A frase original é \"Ameixa Banana amora goiaba banana melancia melão BANANA maçã banana\" e foi alterada para : <br>" + novoTxt;
</script>


Modificador /texto/g

O modificador /texto/g é usado para verificar uma correspondência global, ou seja, ele não na para primeira vez que encontra a palavra idêntica, ele continua executando o método para todas as palavras, entretanto considera diferenças entre letras maiúsculas e minúsculas.

Mostrar Código

<div id="replace2"></div>
<script>
var txt = "Ameixa Banana amora goiaba banana melancia melão BANANA maçã banana";
var novoTxt = txt.replace(/banana/g, "QUEIJO");
document.getElementById("replace2").innerHTML = "A frase original é \"Ameixa Banana amora goiaba banana melancia melão BANANA maçã banana\" e foi alterada para : <br>" + novoTxt;
</script>

Colchetes []

Em RegEx os colchetes são usados para encontrar um grupo de caracteres em específico. por exemplo, a[abc]d pode ser a palavra aad, abd ou acd.

Expressão Descrição
[abc] Pode ser qualquer dos caracteres que estão dentro dos colchetes.
[^abc] Pode ser qualquer dos caracteres que NÃO estejam dentro dos colchetes.
[0-9] Pode ser qualquer dos caracteres dentro do intervalo - definido, pode ser [a-d], ou [a-z] ou ainda [a-zA-Z].
[^0-9] Pode ser qualquer dos caracteres que não estejam dentro do intervalo - definido
[a|b] Poder ser o caractere a ou b.

Mostrar Código

<div id="replace6"></div>
<script>
var txt = "190 10 170 1270 1765 90 0 140";
var novoTxt = txt.replace(/1[0-9]0/igm, "SUB");
document.getElementById("replace6").innerHTML = "\"190 10 170 1270 1765 90 0 140\" <br>" + novoTxt;
</script>

Metacaracteres \

Metacaracteres são caracteres que possuem um maior significado que caracteres simples. Normalmente um metacaractere vem precedido do caractere de contra-barra \. Para ver outros metacaracteres e mais exemplos acesseexpressões Regulares

Metacaractere Descrição
. Pode ser qualquer caractere exceto de quebra de linha ou fim de linha.
\w Encontra um caractere de letra
\W Encontra um caractere que não seja uma letra
\d Encontra um caractere que seja um digito
\D Encontra um caractere que não seja um digito
\s Encontra um caractere que seja um espaço em branco
\S Encontra um caractere que não seja um espaço em branco
\b Encontra um caractere que seja o início ou final de uma palavra
\B Encontra um caractere que não seja o início ou final de uma palavra
\0 Encontra um caractere que seja nulo
\n Encontra um caractere que seja uma quebra de linha
\t Encontra uma tabulação horizontal
\v Encontra uma tabulação vertical

Exemplo explicado... vamos procurar uma palavra que começa com "te" que tenha venha seguido de dois quaisquer caracteres ".." e tenha uma letra no final "\w", onde não importa a letras maiúsculas e que sejam todas as palavras, por isso chegamos em: /te..\w/igm.

Mostrar Código

<div id="replace7"></div>
<script>
var txt = "texto 73x70 tebcd tebc0";
var novoTxt = txt.replace(/te..\w/igm, "SUB");
document.getElementById("replace7").innerHTML = "\"texto 73x70 te370\" <br>" + novoTxt;
</script>

Quantificadores * ^ ? + $ !

Os caracteres * ^ ? + $ ! são usados para quantificar nossas expressões regalares ou mesmo palavras. Por exemplo, queremos todas as palavras que começem com a e terminem com z, ao invés de escrever infinitamente a..........z podemos usar o quantificador * ficando a.*z.

Quantificador Descrição
n+ Corresponde a uma ou mais vezes o valor de n
n* Corresponde a nenhuma ou mais vezes o valor de n
n? Corresponde a nenhuma ou apenas uma vez o valor de n
n{x} Corresponde a exatamente x ocorrências do valor de n
n{x,y} Corresponde a um intervalo de pelo menos x até y ocorrências do valor de n
n{x, } Corresponde a pelo menos x ou mais ocorrências do valor de n
n$ Corresponde a qualquer valor que contenha o valor de n no final
^n Corresponde a qualquer valor que não contenha o valor de n
?=n Corresponde a qualquer string seguida pelo valor de n
?!n Corresponde a qualquer string que não esteja seguida pelo valor de n

Podemos usar parênteses para agrupar expressões para aplicar os quantificadores assim como os modificadores de colchetes.Vamos selecionar todas as palavras que começam por a e tenham pelo menos 3 b's. Observe um possível erro. /ab{3,}/igm

Mostrar Código

<div id="replace8"></div>
<script>
var txt = "abbbc abbc abc acb abbbb bbbbb";
var novoTxt = txt.replace(/a(b){3,}/igm, "SUB");
document.getElementById("replace8").innerHTML = "\"abbbc abbc abc acb abbbb bbbbb\" <br>" + novoTxt;
</script>


Vamos selecionar todas as palavras que começam por a e terminam com c. Observe o erro! Selecionamos toda a string pois especificamos qualquer caractere entre a e c, podendo ser espaços. a(.)*c

Mostrar Código

<div id="replace9"></div>
<script>
var txt = "abbbc abbc abc abbbbbbbb ac acccccb";
var novoTxt = txt.replace(/a(.)*c/igm, "SUB");
document.getElementById("replace9").innerHTML = "\"abbbc abbc abc abbbbbbbb ac acccccb\" <br>" + novoTxt;
</script>


Para corrigir isso precisamos especificar que entre a e c podem haver quaisquer caracteres exceto espaço em branco! Com isso devemos voltar em colchetes especificar qualquer caractere [.] exceto o espaço em branco [^\w] ficando : /a[.^\w]*c/igm

Mostrar Código

<div id="replace10"></div>
<script>
var txt = "abbbc abbc abc abbbbbbbb ac acccccb";
var novoTxt = txt.replace(/a[.^\w]*c/igm, " - SUB - ");
document.getElementById("replace10").innerHTML = "\"abbbc abbc abc abbbbbbbb ac acccccb\" <br>" + novoTxt;
</script>
