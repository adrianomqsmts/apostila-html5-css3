Como Usar o JavaScript?

Para usar o JavaScript precisamos declarar o elemento HTML <script></script> em qualquer parte do <body> ou <head> e dentro dele usar a sintaxe da linguagem. Podemos também usar um arquivo externo com a extensão .js e importá-lo para dentro do projeto com o atributo src="caminho" da tag inicial do elemento <script src="caminho"></script> dentro do <head.

<html>
<head>
<script src="caminho"></script>
<script>CÓDIGO 1</script>
</head>
<body>
<script>CÓDIGO 2</script>
</body>
</html>

As vantagem de se usar o arquivo externo é a possibilidade de aplicar as mesmas funcionalidades a diferente páginas HTML, deixar a estrutura HTML mais limpa, além de facilitar na manutenção das páginas.

O que é o JavaScript?

Basicamente o JS [e uma linguagem de programação que permite a você criar conteúdo que se atualiza dinamicamente em uma página HTML, além de possibilitar controlar multimídias, transições, layout, estilo e tudo mais...

Antes de começarmos...

Os navegadores possuem um console que é muito útil para ver erros e testar funcionalidades JS. O comando básico que a maioria dos navegadores usam é CTRL + SHIFT + I, em seguida procure pela aba escrita console. Muitos exemplos usarão essa interface para demostrar seus conteúdos.
