Math em JavaScript

O objeto Math é usado para realizar os cáculos mais comuns do mundo da matemática como constantes, arredondamentos, seno, cosseno, etc.

Método Descrição
Math.abs(x) Retorna o valor absoluto de x
Math.acos(x) Retorna o arco cosseno de x, em radiano
Math.acosh(x) Retorna o arco cosseno hiperbólico de x
Math.asin(x) Retorna o arco seno x, em radiano
Math.asinh(x) Retorna arco seno hiperbólico de x
Math.atan(x) Retorna o arco tangente de x que é um valor entre -PI/2 e PI/2 em radiano
Math.atan2(y, x) Retorna o arco tangente do quociente entre os valores de y e x
Math.atanh(x) Retorna o arco tangente hiperbólico de x
Math.cbrt(x) Retorna a raiz cúbica de x
Math.ceil(x) Retorna o valor de x arredondado para cima (mesmo que seja 1.1 irá virar 2).
Math.cos(x) Retorna o cosseno de x (x em radiano)
Math.cosh(x) Retorna o cosseno hiperbólico de x
Math.exp(x) Retorna o valor de E elevado a x (Ex )
Math.floor(x) Retorna o valor arredondado para baixo do valor de x (mesmo que seja 1.9 será 1)
Math.log(x) Retorna o logaritmo natural de x
Math.max(x, y, z, ..., n) Retorna o maior valor entre os valores especificados
Math.min(x, y, z, ..., n) Retorna o menor valor entre os valores especificados
Math.pow(x, y) Retorna o valor de x elevado a y (xy )
Math.random() Retorna um valor aleatório entre 1 e 0
Math.round(x) Retorna um valor arredondado mais próximo de x (1.5+ será 2, enquanto 1.49- será 1)
Math.sin(x) Retorna o seno de x (x em radiano)
Math.sinh(x) Retorna o hiperbólico seno de x
Math.sqrt(x) Retorna a raiz quadrada de x
Math.tan(x) Retorna o tangente do angulo x
Math.tanh(x) Retorna a tangente hiperbólica de x
Math.trunc(x) Retorna a parte inteira de um número decimal. (1.89237176 vira 1)

Mostrar Código

<div id="math"></div>
<script>
var a = 36;
document.getElementById("math").innerHTML = "Raiz quadra de 36 : " + Math.sqrt(a) + "<br>" +
"36 elevado a 2: " + Math.pow( a, 2 ) + "<br>" +
"o valor absoluto de -10: " + Math.abs(-10) + "<br>" +
"Um valor aleatório ente 1 e 0: " + Math.random() + "<br>" +
"O valor arredondado de 1.782: " + Math.round(1.782) + "<br>" +
"O valor truncado de 1.782: " + Math.trunc(1.782) + "<br>" +
"O maior valor entre 1, 10, 9, 7 e 11 : " + Math.max(1, 10, 9, 7, 11) + "<br>" +
"O menor valor entre 1, 10, 9, 7 e 11 : " + Math.min(1, 10, 9, 7, 11) + "<br>" +
"O valor truncado de 1.782: " + Math.trunc(1.782) + "<br>" +
"A raiz cúbica de 36 : " + Math.cbrt(a);
</script>

Valores aleatórios

Para retornar valores aleatórios, por exemplo, entre 1 e 100, podemos multiplicar o valor do método Math.random() * 100, arredondar ou truncar esse valor e somar 1. Se quisermos um valor entre 1 e 50, basta multiplicar por 50, enquanto para um valor entre 1 e 10 multiplicamos por 10...

Mostrar Código

<div id="random"></div>
<script>
document.getElementById("random").innerHTML = "Valor aleatório entre 1 e 100: " + (Math.floor(Math.random() * 100) + 1);
</script>

Valores máximo e mínimos de um array

Podemos usar os métodos Math.max.apply(null,array) e Math.min.apply(null,array) para encontrar os valores máximos e mínimos de um array. Esse método converte o array de [1,2,3,4..] para 1,2,3,4.. aplicando em seguida o método padrão de máximo e mínimo que precisa de apenas números separados por vírgulas como parâmetros.

Mostrar Código

<div id="max-min"></div>
<script>
var array = [100, 50, 60, 10, 80, 0, 1, 101 ];
var max = Math.max(array);
var min = Math.min(array);
document.getElementById("max-min").innerHTML = "O array é: " + array.toString() + "<br>" +
"O menor valor é: " + min + "<br>" +
"O maior valor é: " + max;
</script>
