Operadores em JavaScript

Podemos começar com os operadores matemáticos básicos que são multiplicação *, divisão /, subtração - e soma +. Veremos mais à frente que o operador de soma (+) pode assumir outras funcionalidades.

Mostrar Código

<div id="conta1"></div>
<script>
document.getElementById("conta1").innerHTML = 10 + 10;
</script>

A forma mais comum de formar uma operação matemática é se utilizando de variáveis (pois normalmente é de onde vem os valores que o usuário digita).

Mostrar Código

<div id="conta2"></div>
<script>
var a = 10;
var b = 20;
document.getElementById("conta2").innerHTML = a - b;
</script>

Abra o console e veja o resultado do código abaixo.

Mostrar Código

<script>
var a = 5;
var b = 2;
console.log(a * b);
console.log(a / b);
</script>

Operadores matemáticos II

Temos ainda outros operadores matemáticos um pouco mais complexo, porém comuns na comunidade da programação.

Operador Descrição Exemplo
+ Adição a + b
- Subtração a - b
* Multiplicação a * b
** Exponenciação a ** b
/ Divisão a / b
% Módulo (Resto de uma divisão inteira) a % b
++ Incrementar 1 do valor da variável a++
-- Decrementar 1 do valor da variável a--

Abra o Console e veja os resultados para os operadores II

Mostrar Código

<script>
var a = 5;
var b = 2;
console.log("Operadores II - a = 5 e b = 2");
console.log("Soma: " + a + b);
console.log("Subtração: " + (a - b));
console.log("Multiplicação: " + (a * b));
console.log("Divisão: " + (a / b));
console.log("Módulo: " + (a % b));
console.log("Exponenciação: " + (a ** b));
console.log("Incremento: " + (a++));
console.log("Decremento: " + (a--));
</script>

Notou algo de estranho no incremento e decremento? Então, isso acontece pois primeiro é verificado o valor de a e depois o valor e incrementado, por isso no decremento ocorre o valor 6. Se tentássemos exibir novamente a variável a, iria nos retornar o valor 5, pois depois de exibir o valor 6 foi realizado o decremento.

Operadores Curtos

Podemos encurtar nossas expressões matemáticas com os operadores abaixo, mas lembre-se de alguns pontos importantes... O primeiro é que a variável contraída deve inicialmente sempre tem um valor inicial, geralmente 0. E normalmente a expressão encurtada tem a seguinte ocorrências: a = a + b, ou seja, a variável que irá receber o valor, deve estar contida no corpo da expressão.

Operador Forma Original Forma contraída
+= x = x + y x += y
-= x = x - y x -= y
*= x = x * y x *= y
/= x = x / y x /= y
%= x = x % y x %= y
**= x = x ** y x **= y

Mostrar Código

<div id="conta6"></div>
<script>
var a = 10;
document.getElementById("conta6").innerHTML = (a += 5);
</script>

Operadores de Comparação

Os operadores de Comparações são usados para comprar o valor e/ou o tipo da variável retornando um valor verdadeiro ou falso.

Operador Descrição/th>
== Igual à
=== Igual à e de mesmo tipo
!= Não é igual
!== Não é igual e não é do mesmo tipo
> Maior que
< Menor que
>= Maior e igual à
<= Menor ou igual à
? Operador Ternário
&& Operador Lógico AND (e)
|| Operador Lógico OR (ou)
! Operador Lógico NOT (negação)

Estes operador ficam mais claros quando se formula uma frase. Por exemplo: A variável a é igual à variável b e menor que a variável c? Dependo dos valores de a, b ou c essa pergunta pode retornar verdadeiro ou falso (true or false).

Mostrar Código

<div id="conta7"></div>
<script>
var a = 10;
var b = 10;
var c = "10" ;
var d = 10;
document.getElementById("conta7").innerHTML =
"a igual á b? " + (a == b) +
"<br> a igual á c? " + (a == c) +
"<br> a igual á b e do mesmo tipo? " + (a === b) +
"<br> a igual á c e do mesmo tipo? " + (a === c) +
"<br> a igual á b e menor que d? " + (a == c && a < d) + "<br>";
</script>

Estas são apenas algumas das possibilidades de trabalhar com lógica em JS, mais à frente veremos seus uso cada vez mais contextual. Lembrando que uma lógica sempre irá retornar verdadeiro (true) ou falso (false)

Ordem das expressões

Podemos alterar a ordem em que as expressões são calculadas, para isso deve se lembrar de matemática básica e o uso de parênteses para alterar a ordem. Em matemática, temos a seguinte ordem para calcular:

  1. 1º Parêntesis;
  2. 2º Expoentes;
  3. 3º Multiplicações e Divisões; (da esquerda para a direita)
  4. 4º Somas e Subtrações. (da esquerda para a direita)

Mostrar Código

<div id="ordem"></div>
<script>
var soma1 = 1 + 10 * 10 ;
var soma2 = (1 + 10) * 10;
document.getElementById("ordem").innerHTML = "Resultado da soma1 é: " + soma1 + "<br> o resultado da soma2 é: " + soma2;
</script>

Assim como em lógica também existem ordem de precedência e podemos alterá-la usando parênteses. Para saber a precedência completa em JS acesse: Precedência de operadores JS.
