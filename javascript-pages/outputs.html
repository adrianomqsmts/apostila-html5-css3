Outputs com JavaScript?

Existem 4 principais formas de exibir algum conteúdo utilizando JS.

  • A função document.write() que escreve no HTML e deve ser usada somente para teste;
  • A função window.alert() que abre um caixa de mensagem para exibir o conteúdo;
  • A função console.log() que exibe o conteúdo na saída do console do navegador e
  • E a funcionalidade innerHTML que deve ser usada para inserir um conteúdo em um elemento HTML

Mostrar Código

<script>
document.write('Oi! Eu sou um texto, inserido usando o <strong>documente.write</strong>');
</script>


Abra o console e veja a mensagem... Lembre-se do comando CTRL + SHIFT + I.

Mostrar Código

<script>
console.log("Oi! Está me vendo no console?");
</script>


Podemos usar o innerHTML para escrever em um elemento HTML. Primeiramente iremos abstrair alguns detalhes, mas para usar a função precisamos primeiro pegar um elemento para escrever nele. Importante LEMBRAR-SE que sempre que usar o innerHTML em um elemento, deve-se usá-lo após a criação do elemento!

Mostrar Código

<div id="teste"></div>
<script>
document.getElementById("teste").innerHTML = "Olhe! Eu escrevi um conteúdo dentro do elemento!!!");
</script>


Por fim, podemos ainda abrir uma pequena janela de alerta para exibir uma mensagem.

Mostrar Código

<script>
window.alert("Oi! Estou abrindo uma janelinha!!!");
</script>
