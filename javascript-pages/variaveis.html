Variáveis JavaScript

Caso não saiba, variáveis são como com container para guardar diferentes tipos de dados, como textos, caracteres, números, etc. O conceito mais básico para declarar uma variável é a palavra-chave var. Veja abaixo um exemplo simples.

var x = 10;
var y = 20;
var z = x + y;

Outra forma de declarar um variável é usando a palavra-chave const que são variáveis que não podem ter seus valores alterados ou que são contantes como o valor do PI. Existem por fim a palavra-chave let que declara variáveis de escopo restrito, e você pode ver mais sobre o seu uso em funções

Regras de declaração de variáveis

Existem algumas regras na criação do nome de uma variável (identificador).

  • Um nome de variável deve sempre começar por uma letra. (Eles podem começar por _ e $, mas somente casos específicos)
  • Não utilize símbolos especiais tais como:(! @ # - % ^ & * : ; " ', . / ? + = ~`) na criação de do nome de uma variável.
  • Procure escrever os nomes das variáveis sempre em minúsculo uma vez que Javascript é sensível a letras maiúsculas e minúsculas, (ou seja Abc é diferente de abc)
  • Procure dar nomes as variáveis que lembrem o seu conteúdo como nome, idade, sexo, altura, etc.
  • Palavras reservadas (como palavras-chave JavaScript e var) não podem ser usadas como nomes
  • Variáveis de nome duplo a separação é recomendada por letra maiúscula como em primeiroNome e segundoNomeGrande. Esta técnica é conhecida como camelCase

Operador de atribuição

Parar atribuirmos um valor à uma variável devemos usar o caractere = que irá ligar o nome da variável ao valor atribuído. Procure ao final de cada declaração colocar o sinal de ; para delimitar os comandos (mesmo que o ; ao final em JS não seja necessário).

var x = 10;
var y = "Meu nome";
var z = 10.6;

Tipos de Dados em JS

Não é preciso especificar o tipo de variável, pois ao criarmos uma variável e atribuir um conteúdo do tipo texto, e em seguida, na linha de baixo, usarmos a mesma variável para atribuir um conteúdo numérico, o próprio navegador interpreta a instrução e converte o tipo normalmente sem nenhum problema. Mas isso não é simples assim, pois em determinados momentos precisaremos forçar a conversão do tipo de dados como vermos mais tarde.

var x = 10; //x agora é numérico
var x = "Meu nome"; //x agora é uma string

Os principais tipos de dados em JS é o tipo string (texto), number (número), object (objetos), booleans (condicional true/false (verdadeiro ou falso)), arrays (matrizes), undefined (indefinido) e null (nulo). No decorrer das páginas da apostila veremos os principais detalhes sobre cada um destes tipos de dados.

A principal diferença entre uma tipo de dado undefined e null, é que a variável undefined nunca teve um valor atribuído para ela, enquanto a variável null era um variável que continha um valor e agora não possui mais, e está vazia.

var x; // x agora está undefined
var x = 10; // x agora é do tipo numérico
var x = null; // x agora está sem valor
